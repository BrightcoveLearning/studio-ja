クイック検索を使用する

Analytics モジュールのクイック検索機能を使用して、特定の動画を検索できます。検索条件を入力すると、クイック検索では、動画名、短い説明、詳細説明、タグ、動画 ID、およびカスタムテキストフィールドを検索します。検索結果が検索フィールドの下に表示されます。

ご了承ください削除されました動画は検索機能では検索できませんが、分析は行っています。パフォーマンスレポートを展開/ナビゲートし、ブラウザ検索機能を使用するか、レポートをダウンロードしてスプレッドシートアプリで検索することで、それらを見つけることができます。

マッチへのリンクをクリックすると、それに関連したレポートが開きます。たとえば、アカウントのエンゲージメントを確認し、[クイック検索] を使用して特定の動画を検索する場合、動画へのリンクをクリックすると [エンゲージメントレポート] が開きます。パフォーマンスレポートを表示している場合は、リンクを検索してクリックすると、パフォーマンスレポートが開きます。

基本検索

修飾子なしで検索値を指定すると、次のフィールドに対してテキスト検索が実行されます。

  • 名前
  • 短い説明
  • 長い説明
  • タグ
  • ビデオ ID
  • 参照ID
  • すべてのカスタムフィールド

複数の単語を一致させる場合は、単語を引用符で囲みます。

高度な検索

Analytics モジュールでは、特定のフィールドを検索したり、日付に基づいて検索したり、特定の用語を含めたり除外したりする機能も用意されています。

特定のフィールドの検索

指定した値を特定のフィールドで検索するには、次の構文を使用します。

  • field:value to search

次のフィールドがサポートされています。

フィールド サポートされている値
name 文字列または引用符で囲まれた文字列
text 文字列または引用符で囲まれた文字列 ( name、short description、内の値long description )
tags 文字列または引用符で囲まれた文字列
reference_id 文字列または引用符で囲まれた文字列
state ACTIVE , INACTIVE , DELETED , PENDING
updated_at 日付形式の特定の日付を検索するを参照してください。
created_at 日付形式の特定の日付を検索するを参照してください。
published_at 日付形式の特定の日付を検索するを参照してください。
complete trueまたはfalse

完全一致をしたり、複数の単語を一致させたい場合は、単語を引用符で囲みます。

  • "live"
  • "live streaming"

これは、特定のフィールドを検索するときにも機能します。

  • name:"live"
  • name:"live streaming"

特定の日付を検索

日付に依存する検索フィールドには、日付範囲の値が使用されます。日付を指定する複数の形式がサポートされています。

[形式] 意味深
2013-08-01 yyyy-mm-dd形式の日付
2013-08-01T 00:00:00 Z UTC形式の時刻
-1d 現在の時刻から 1 日を引いた ( 相対日付を参照 )
+1d 現在の時刻に 1 日を加えたもの ( 相対日付を参照 )
 

相対日付

相対日付の場合、方向 (+ または-) の後に数字、デュレーション (またはその略語) が続きます。法的期間は、分（m）、時間（h）、日（d）

例:

  • -1day
  • -2days
  • +4hours
  • -60minutes
  • -1d
  • -2d
  • +4h
  • -60m

日付範囲

検索するフィールドが日付の場合は、検索する日付の範囲を指定できます。

  • updated_at:2013-08-01T00:00:00Z..2013-10-08T00:00:00Z

これにより、2013 年 8 月 1 日から 2013 年 10 月 8 updated_at日の間の値を持つすべての動画が検索されます。

終了範囲を開く

特定の時刻までのすべての日付を一致させる場合、または特定の時刻以降のすべての日付に一致する場合は、範囲の一端をオフにします。たとえば、過去 2 日間に変更されたすべての動画を検索するには、次のようにします。

  • q=updated_at:-2days..

2013 年 8 月 11 日より前に変更されたすべての動画を検索します。

  • q=updated_at:..2013-08-11T00:00:00Z

 

特定の用語を含める、除外する

用語は、必須（返された動画が一致しなければならない）または禁止（返された動画が一致しないこと）としてマークできます。これは、+-用語の直前にまたは記号で制御されます。

例 意味深
+animals 動画には、「動物」' という用語を含める必要があります。nameshort descriptionまたはlong description
-animals 動画には、「動物」' という用語を含めないでください。nameshort description、またはlong description
+name:animals 動画の名前には「動物」' という用語を含める必要があります。
-name:animals 動画の名前に「動物」' という用語を含めることはできません。
