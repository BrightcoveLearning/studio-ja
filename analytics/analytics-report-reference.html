Video Cloud Analyticsモジュールは、次のレポートを提供します。

  • パフォーマンス
  • エンゲージメント
  • 現在の傾向(Trending Now)
  • カスタム

次のセクションでは、これらの各レポートの概要について説明します。

パフォーマンスレポート

パフォーマンスレポートには、アカウントの動画の再生回数、動画のインプレッション数、再生率、視聴時間、エンゲージメントスコアなど、アカウント内の動画に関するパフォーマンスが表示されます。パフォーマンス・レポートでは、データを次の次元に分けることができます。

  • 動画 -動画のパフォーマンス指標を表示します。
  • プレーヤー -プレーヤーのパフォーマンス指標を表示します。
  • トラフィックソース -トラフィックのソースに基づいてパフォーマンス指標を表示します。
  • ドメイン名 -動画が視聴されたときに視聴者がいたドメインに基づいてパフォーマンス指標を表示します。プレーヤーが iframe 内にある場合、表示ドメインは iframe 内のページの URL として記録されます。(Analytics API destination_domainのディメンションに相当します。)
  • 国 -動画が視聴された国に基づいてパフォーマンス指標を表示します。
  • デバイス -各デバイスタイプのパフォーマンス指標を表示します。-ビデオを表示するために使用されたデバイスオペレーティングシステムの組み合わせ
  • デバイスタイプ: デバイスタイプ別にパフォーマンス指標を表示します。
  • デバイスのOS -デバイスのオペレーティングシステム別のパフォーマンス指標を表示します
  • 機器メーカー -デバイスの製造元別のパフォーマンス指標を表示します
  • ブラウザのタイプ: ブラウザのタイプ別にパフォーマンス指標を表示します。
  • 日 - 日別のパフォーマンスデータを表示します

パフォーマンスレポートの詳細については、「パフォーマンスレポートのリファレンス」を参照してください。

エンゲージメントレポート

エンゲージメントレポートには、アカウント内の動画のエンゲージメントが表示されます。これには、エンゲージメントスコアと、動画の 1%、25%、50%、75%、100% のポイントにおける視聴回数が含まれます。エンゲージメントレポートでは、データをディメンションに分けることができます。

  • 動画 -動画のエンゲージメント指標を表示
  • プレイヤー -プレイヤーのエンゲージメント指標を表示します。
  • 日 -エンゲージメント指標を日ごとに表示します

エンゲージメントレポートの詳細については、エンゲージメントレポートリファレンスをご覧ください。

現在の傾向 (Trending Now)

現在の傾向(Trending Now)レポートには、過去 1 時間の同時視聴者数を示すグラフが表示されます。現在の傾向(Trending Now)レポートを使用して、VOD、ライブ、または両方のタイプのビデオのデータを表示できます。

現在の傾向(Trending Now) レポートの詳細については、「現在の傾向(Trending Now)レポートのリファレンス」を参照してください。

カスタムレポート

Analyticsモジュールを使用して、アドホックにカスタム分析レポートを作成できます。レポートは、コンマ区切り（.csv）形式またはMicrosoft Excel（.xlsx）形式でダウンロードできます。

注:カスタムレポートは、Enterprise および Pro パブリッシャーが使用できます。

カスタムレポートの作成については、Analytics モジュールでのカスタム Analytics レポートの作成を参照してください。

レポートデータのエクスポート

レポートデータのダウンロードは、コンマ区切り形式とExcel形式で可能です。
