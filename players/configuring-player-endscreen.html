プレーヤーは、ビデオが終了したときに終了画面が表示されるように設定できます。終了画面は、再生アイコンと共有アイコンを表示するように構成することも、HTML を含めるようにカスタマイズすることもできます。カスタム終了画面を以下に示します。

アイコンが表示され、視聴者は動画をリプレイしたり、人気のソーシャルメディアサイトに動画を共有することができます。

注: プレーヤーのエンドスクリーンが構成されている場合、プレーヤーがネイティブフルスクリーンモードの場合、iOSデバイスには表示されません。

プレーヤーの終了画面を構成するには、次の手順に従います。

  1. ナビゲーションヘッダーの [ Players ] をクリックして [Players] モジュールを開きます。
  2. 設定するプレーヤーのリンクをクリックします。
  3. 左側のナビゲーションメニューで [ 画面の終了] をクリックします。
  4. [ プレーヤーの終了画面 ] オプションを選択します。次のオプションを使用できます。
    • なし-終了画面は表示されません
    • ソーシャル -ビデオが終了したときにソーシャル共有ダイアログを表示する
    • カスタム -ビデオが終了すると、カスタム終了画面を表示します
注:プレイヤーに加えた変更は、公開されているプレイヤーに表示されるまでに最大 5 分かかる場合があります。

ソーシャル終了画面

Socialオプションを選択すると、ビデオが終了したときにソーシャル共有ダイアログが表示されます。

ソーシャル共有の終了画面を使用するには、プレーヤーのソーシャル共有を有効にする必要があります。ソーシャル ] オプションを選択すると、ソーシャル共有が自動的に有効になります。表示されるソーシャル共有オプションは、ソーシャル設定の一部として構成された内容に基づいています。ソーシャル設定の詳細については、「Player  のソーシャルプロパティの設定」を参照してください。

カスタム終了画面

「カスタム」オプションを選択すると、カスタム HTML を使用して、再生アイコンと共有アイコンの上に追加のテキストまたは画像を表示できます。カスタム終了画面を構成するには、[ カスタム] を選択し、[カスタム HTML で貼り付け] を選択して、[ カスタム HTML]プロパティに表示します。

注:ビデオは、カスタム HTML に <video> html 要素を追加することによって使用できます。
注:共有アイコンを終了画面に表示するには、共有を有効にする必要があります。
注:「カスタム」オプションが選択され、HTML が指定されていない場合、ビデオの終了時に再生ボタンが表示されます。
