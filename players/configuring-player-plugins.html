Players モジュールの Plugins オプションを使用すると、プレーヤープラグイン、スタイルシートを追加したり、スクリプトを Brightcove Player に追加したりできます。[プラグイン] ページには、プレーヤー用に構成されているすべてのプラグインが一覧表示されます。Brightcoveが提供するプラグインは表示されません（つまり、ソーシャル、オーディエンスなど）。

注:プラグイン機能は、Video CloudEnterpriseのお客様が利用できます。Video Cloud アカウントのアップグレードに興味がある場合は、Brightcove にお問い合わせください。

プレーヤープラグインの構成

プラグインは通常、次の目的で開発されます。

  • プレイヤーのデフォルトの動作を変更する
  • プレーヤーに機能を追加する
  • プレイヤーの外観をカスタマイズする

独自のカスタムプラグインの作成については、ステップバイステップを参照してください。プラグイン開発。

プレーヤーのプラグインを設定するには、次の手順に従います。

  1. ナビゲーションヘッダーの [ Players ] をクリックして [Players] モジュールを開きます。
  2. 設定するプレーヤーのリンクをクリックします。
  3. 左側のナビゲーションメニューで [ プラグイン ] をクリックします。
  4. [ プラグインを追加]をクリックし、プラグインの種類をクリックします。
    • カスタムプラグイン -あなたまたは第三者が開発したプラグイン
    • Brightcoveプラグイン -Brightcoveによって開発および提供されたプラグイン
  5. にとってBrightcoveプラグイン、 を選択Brightcoveプラグインリストから。利用可能な Brightcove プラグインはすべて、リストには使用できません。Brightcove が提供するすべてのプラグインの一覧については、「ブライトコーブ Player プラグイン」トピックを参照してください。
  6. [ カスタムプラグイン]で、次のように入力します。
    • プラグイン名 -プラグインJavaScriptファイルで定義されている通り
    • JavaScript URL -プラグインコードを含むファイルへの URL を指定します。
    • CSS URL -プラグインで使用される CSS コードを含むファイルへの URL を指定します。有効な CSS を指定してください。無効なCSSを指定すると、プレビューウィンドウにエラーが表示されます。
  7. Options (JSON)フィールドを使用して、両方のプラグインタイプの JSON 形式のオプションを貼り付けることができます。Brightcove プラグインの場合、使用可能なオプションの概要はプラグインのドキュメントに記載されています。カスタムプラグインの場合、オプションの使用方法はプラグインの JavaScript コードに表示されます。
  8. [保存] をクリックして、変更を保存します。
注:プレイヤーに加えた変更は、公開されているプレイヤーに表示されるまでに最大 5 分かかる場合があります。

プラグインのプロパティを編集するには、プラグイン名をクリックします。プラグインを削除するには、削除アイコンをクリックします（）。

スタイルシートの追加

CSS ファイルをアップロードして、プレーヤーの外観をカスタマイズできます。

プレーヤーに CSS ファイルを追加するには、次の手順を実行します。

  1. ナビゲーションヘッダーの [ Players ] をクリックして [Players] モジュールを開きます。
  2. 設定するプレーヤーのリンクをクリックします。
  3. 左側のナビゲーションメニューで [ プラグイン ] をクリックします。
  4. [ スタイルシート]リンクをクリックします。
  5. [ スタイルシートを追加] をクリックします。
  6. CSSファイルへのURLを提供します。
  7. [保存] をクリックして、変更を保存します。
注:プレイヤーに加えた変更は、公開されているプレイヤーに表示されるまでに最大 5 分かかる場合があります。

ファイルパスを編集するには、ファイル名をクリックします。ファイルを削除するには、削除アイコン（）。

スクリプトの追加

スクリプトファイルをアップロードして、プレーヤーの外観をカスタマイズできます。

注:JavaScriptを使用してプレーヤーの動作を変更する場合は、プラグインを使用する必要があります。ここで JavaScript ファイルをアップロードしても、プレーヤーには影響しません。

スクリプトファイルをプレーヤーに追加するには、次の手順に従います。

  1. ナビゲーションヘッダーの [ Players ] をクリックして [Players] モジュールを開きます。
  2. 設定するプレーヤーのリンクをクリックします。
  3. 左側のナビゲーションメニューで [ プラグイン ] をクリックします。
  4. [ スクリプト]リンクをクリックします。
  5. [ スクリプトを追加] をクリックします。
  6. JavaScriptファイルへのURLを提供します。
  7. [保存] をクリックして、変更を保存します。
注:プレイヤーに加えた変更は、公開されているプレイヤーに表示されるまでに最大 5 分かかる場合があります。

ファイルパスを編集するには、ファイル名をクリックします。ファイルを削除するには、削除アイコン（）。

プラグインコードの場所

上記のようにプラグインが Brightcove 提供のプラグインでも、自分で作成したカスタムプラグインでも、プレーヤーに追加された場合、または Player Management API を使用すると、プラグインはプレーヤーコード自体の一部になります。プラグインコードは、インターネット上の格納場所から取得され、プラグインが追加された特定のプレーヤーのコードの一部になります。これをさらに説明するために、Advanced (ページ内) プレーヤーの実装を使用すると、<script>次のようなタグが表示されます。

	<script src="//players.brightcove.net/1507807800001/default_default/index.min.js"></script>

これは、Brightcove プレーヤーを定義するコードです。このファイルには、プレーヤーに追加したプラグインのコードが含まれています。プラグインコードは、標準 (iframe) プレーヤーの実装にも含まれています。

独自のプラグインを開発し、インターネットストレージの場所でソースコードを変更する場合は、新しいプラグインコードは JavaScript ファイルの一部にならないため、プレーヤーを再公開してください。

プラグインの作成については、ステップバイステップを参照してください。プラグイン開発開発者ドキュメント。
