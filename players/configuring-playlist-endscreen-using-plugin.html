NS プレイリスト-エンドスクリーンプラグインを使用すると、動画の再生リストをプレーヤーのエンドスクリーンに表示できます。次に、視聴者はビデオのサムネイル画像をクリックして別のビデオを見ることができます。このプレーヤーでは、プレイリストの終了画面が構成されています。

プレーヤーのエンドスクリーンの他のオプションについては、を参照してください。プレーヤーのエンドスクリーンの構成。

注:プレイリストの終了画面をプログラムで設定する方法については、開発者向けドキュメントを参照してください。

プレイリスト終了画面プラグインの構成

Playersモジュールを使用してplaylist-endscreenプラグインを実装するには、次の手順に従います。

  1. Playersモジュールを開き、プレーヤーを見つけます。
  2. プレーヤーのリンクをクリックして、プレーヤーのプロパティを開きます。
  3. クリックプラグイン左側のナビゲーションで。
  4. クリックプラグインを追加する>Brightcoveプラグイン。
  5. のためにBrightcoveプラグイン、 選択するプレイリスト-エンドスクリーン。
  6. NS オプション（JSON）が含まれています
    • プレイリストID -エンドスクリーンに表示するプレイリストのID
    • 題名（オプション）-ビデオのリストの上に表示するタイトル
    7. {
  "playlistId": "1234567001",
  "title": "Related Videos"
}
  7. [ 保存 ] をクリックし、プレイヤーを公開します。公開されたプレーヤーに変更が表示されるまで、最大5分かかる場合があることに注意してください。
