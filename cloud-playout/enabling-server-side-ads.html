はじめに

クラウドプレイアウトチャネルを収益化する1つの方法は、広告を使用することです。広告を配信する最も信頼できる方法は、サーバー側の広告挿入（SSAI）を使用することです。

アカウントにSSAI機能を追加することに興味がある場合は、アカウントマネージャーに連絡してください。

このトピックでは、クラウドプレイアウトチャネルでSSAIを有効にし、プレロールおよびポストロールの広告挿入を設定する方法について説明します。ミッドロール広告の挿入は異なる働きをするので、それはでカバーされていますミッドロール広告をチャネルに追加する。プレロール、ポストロール、ミッドロールの広告はすべて、VODコンテンツのチャンネルに挿入できます。

SSAI対応チャネルの場合、宛先を広告対応RTPライブイベントに設定する必要があります。

要件

クラウドプレイアウトでサーバーサイド広告挿入（SSAI）を使用するには、次のものが必要です。

  • アカウントは、ブライトコーブのライブおよびライブwith SSAI で有効にする必要があります。
  • チャンネルの宛先として Brightcove Live または S3 を使用する必要があります。

サーバーサイド広告の有効化

チャネルを作成するときに、SSAIを有効にするオプションがあります。これにより、Brightcoveのサーバー側広告挿入テクノロジーを使用してサーバー側広告が表示されます。有効にすることもできます

注意:
  • 広告挿入を使用する場合、チャネルはBrightcoveライブイベントにのみ出力できます。
  • 広告の挿入を有効にすると、チャンネルでライブ入力が使用されない場合があります。

広告タグと塗りつぶしスレートの設定

アカウント管理の一環として、広告タグと塗りつぶしスレートが設定されていることを確認してください。広告タグの構成については、「サーバー側広告設定の構成」を参照してください。フィルスレートは、ライブストリームの広告ブレークにギャップがある場合に提供できるコンテンツです。フィルスレートの作成については、ライブ設定の管理を参照してください。

注:Cloud Playout は、ライブ設定の一部として構成されたデフォルトの広告スレートを使用します。

プレースホルダー広告休憩動画の作成

どの動画を「プレースホルダ」動画として使用するかを指定する必要があります。「プレースホルダ」動画がプログラムリストに挿入され、広告が配信され、広告の長さが示されます。Brightcove Live でサーバー側広告挿入を使用すると、「プレースホルダ」動画が広告コンテンツに置き換えられるため、この操作を行う必要があります。広告の「プレースホルダ」動画を挿入しないと、実際の動画の一部が広告コンテンツに置き換えられます。広告期間が返された広告よりも短い場合、SSAI は広告スレートを表示します。広告期間が返された広告よりも長い場合は、広告の後に広告スレートが表示されます。

広告休憩時間と同じ動画の長さを持つ動画ソースファイル（2 分、4 分など）を作成/アップロードする必要がある場合があります。彼らは実際にプログラムの一部として再生されないので、これらのビデオの内容は重要ではありません。

ビデオを「プレースホルダー」ビデオとしてマークするには、Media モジュールを使用してこれらのビデオのビデオプロパティを編集し、タグを付けますcp-ad-holder。

チャネルのサーバー側広告の有効化

チャンネルを作成するときは、[ 広告挿入 ] オプションを [ オン] に設定します。

注:既存のチャンネルの場合、広告挿入の設定は変更できません。

プログラムへの広告プレースホルダーの挿入

広告が表示されるプログラムのリストに広告プレースホルダを挿入する必要があります。広告プレースホルダを挿入するには、次の手順に従います。

  1. チャンネル名をクリックして編集します。
  2. 番組一覧で、広告を再生する動画の横にあるチェックボックスをクリックします。
  3. [ 広告の挿入] をクリックし、広告期間を選択します。広告期間のリストは、すべての広告ブレークの「プレースホルダー」動画（タグ付きcp-ad-holder）の長さに基づいて生成されます。
  4. [ OK] をクリックします。選択した長さの「プレースホルダー」ビデオは、選択したビデオの後にプログラムに挿入されます。
    注:「プレースホルダー」として挿入されるビデオしないプログラムの一部として再生されます。この動画は、広告リクエストが行われる場所と広告の長さを示すためにのみ使用されます。
  5. [ 保存 ] をクリックして、プログラムへの変更を保存します。

広告設定の選択

プログラムの再生が開始されたら、[ブライトコーブ Live で表示] をクリックして、ライブモジュールでチャンネルを表示します。プレーヤーの下のドロップダウンを使用して、使用する広告構成を選択できます。

サーバーサイド広告挿入の詳細については、「Video Cloud SSAI の概要」トピックを参照してください。
