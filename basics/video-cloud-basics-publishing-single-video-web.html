前提条件

  • Video Cloud アカウントに動画をアップロードしました
  • プレイヤーが作成されました

ステップ

  1. メディアモジュールを開きます。
  2. ビデオのサムネイルの横にあるチェックボックスをオンにして、公開するビデオを選択します。
  3. [ パブリッシュと埋め込み] > [Web にパブリッシュ] をクリックします。
  4. [プレーヤーの選択]ドロップダウンリストで、使用するプレーヤーを選択します。
    注:Brightcoveのデフォルトプレーヤーは、すべての Video Cloud アカウントにデフォルトで作成されます。
  5. クリックURLプレビュープレーヤーの下で、新しいブラウザタブで開きます。ビデオが表示され、再生できることを確認します。
  6. 埋め込みコードをコピーして Web ページに貼り付けます。
