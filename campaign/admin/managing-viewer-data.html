[ビューアデータの管理] オプションを使用すると、ユーザーを検索し、そのユーザーを削除するか、そのユーザーのデータをエクスポートすることができます。

注意:
  • 通常、マーケティングオートメーションプラットフォームと統合する場合（MAP統合）、データ保持期間の設定を使用して、データを保持する期間を制御します
  • [ビューアデータの管理]メニューにアクセスするには、アカウント管理者である必要があります

ビューアデータを管理するには、次の手順に従います。

  1. キャンペーンモジュールを開きます。
  2. クリックビューアデータの管理左側のナビゲーションで。
  3. 検索するユーザーを入力します。通常、これは次のいずれかの値になります。
    • Eメール -キャンペーントラッキングを使用していて、キャンペーンリードフォームが使用されている場合、またはSSO統合がある場合
    • ユーザー ID -SSO またはカスタムインテグレーションを使用する場合
    • カスタム GUID -匿名ユーザー用
    • 外部 GUID -MAP 統合が使用されているとき
  4. 一致するものが見つかった場合は、次のいずれかのアクションを選択します。
    • 閲覧者データの削除 -ユーザーの閲覧データとリードデータを削除します
    • ビューアーデータのエクスポート -ユーザーのリードと表示イベントデータをエクスポートします。リードデータには、キャンペーンリードフォームに入力されたデータが含まれます。ビューイベントデータには、ビューアのすべてのビューイベントが含まれます。
