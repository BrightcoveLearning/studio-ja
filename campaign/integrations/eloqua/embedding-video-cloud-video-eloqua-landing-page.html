キャンペーンモジュールは、クラウドコンテンツの一種としてビデオクラウドビデオをEloquaランディングページに追加する機能を提供します。EloquaランディングページWYSIWYGエディター内からランディングページにビデオが追加されます。

Eloquaランディングページにビデオを追加するには、次の手順を実行する必要があります。

  1. Eloquaでランディングページを作成する
  2. ビデオとプレーヤーを選択します

Eloquaランディングページの作成

Eloqua ランディングページは、Eloqua から送信された電子メールからクリックスルーした訪問者の目的地として使用できます。ランディングページを作成するには、次の手順に従います。

注:Eloqua ランディング・ページの詳細は、 Oracle Eloqua のドキュメントを参照してください。
  1. VideoCloudアカウントにサインインします。ランディングページで使用する動画があるアカウントを選択します。このアカウントには、キャンペーンモジュールへのアクセス権も必要です。
  2. お客様のEloquaアカウントにサインインします。
  3. アセット > ランディングページに移動し、「ランディングページの作成」をクリックします。
  4. [ 空白の HTML ランディングページ]をクリックし、[ 選択] をクリックします。
  5. 左側のナビゲーションで、[ クラウドコンテンツ] をクリックします。[クラウドコンテンツ]ダイアログが開きます。
  6. Brightcove Videoオブジェクトを HTML ページにドラッグし、<body>タグの間にドロップします。
    注:<body>タグの間にプレイヤーをドラッグ＆ドロップすることで、複数のプレイヤーをランディングページに追加できます。各プレイヤーを追加した後、ランディングページを保存します。

NS クラウドコンテンツの構成できる場所でダイアログが開きますビデオとプレーヤーを選択します。

動画とプレーヤーを選択する

クラウドコンテンツ設定ダイアログは、Video Cloud アカウント、ビデオ、プレーヤーを選択するために使用します。

  1. 右上のドロップダウンリストを使用して、使用する Video Cloud アカウントを選択します。アカウントが選択されると、ダイアログが更新され、選択したアカウントの動画とプレーヤーが表示されます。
  2. ランディングページに表示するビデオを選択します。最新の20本のビデオがリストに表示されます。
  3. 使用するプレーヤーを選択します。キャンペーンが有効なプレイヤーのみが表示されます。
  4. プレイヤーサイズを設定します。
  5. [ 保存 ] をクリックして変更を保存します。
  6. [ X] をクリックして、[クラウドコンテンツ設定] ダイアログボックスを閉じます。HTMLは、プレーヤーを埋め込むためのコードで更新されます。
  7. (オプション)ランディングページに他のオブジェクトを追加します。
  8. [ 保存 ] をクリックして、ランディングページを保存します。
  9. ランディングページに名前を付けて、[ 保存] をクリックします。
