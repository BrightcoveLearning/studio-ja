キャンペーンモジュールは、ビューイベントを追跡する機能を提供します。ビューイベントがキャンペーンモジュールに送信される頻度は、ビデオの長さに基づいて動的に計算されます。キャンペーンは、四分位数ごとに1つずつ、少なくとも4つのイベントを送信します。

注:ビューイベントデータは、[ データのエクスポート]オプションを使用するか、 Audience APIを使用してエクスポートできます。

イベントの表示を表示するには、キャンペーンモジュールを開き、[  最近のすべての活動左側のナビゲーションで。このページには、過去 24 時間の表示イベントアクティビティが表示されます。

NS インタラクションイベントを表示するリンクは次の場合に有効になりますカスタムイベントまたはメトリックビューイベントに添付されます。

このページには、次の情報が表示されます。

  • タイムスタンプ -ビデオビューの日付と時刻
  • ビデオの名前と ID -ビデオ名とビデオクラウドビデオ ID
  • 視聴率 -視聴者がビデオの視聴を停止した時点を、動画の長さで割った値です。例えば、視聴者が 2 分間の動画の最初の 5 秒を視聴し、55 秒のポイントまでスキップし、さらに 5 秒視聴してビデオを停止した場合（またはページを離れた）場合、視聴率は 50% として記録されます。
    注:にとってライブビデオ、キャンペーンで視聴された割合を追跡することはできません。Campaignモジュールはどのくらいの期間続くかを知ることができないため、Campaignモジュールは総視聴時間のみを追跡します。
  • ウォッチ時間 -合計視聴時間
  • 外部 ID -ビューアを識別するために使用される値。たとえば、Eloqua GUID、Marketo GUID、HubSpot GUID、SSO GUID、またはビューアを識別するために使用された別の一意の値です。
  • Status -イベントのステータス
    • 未処理 -イベントがマーケティングオートメーションプラットフォームに同期されていません
    • 同期済み -イベントはマーケティングオートメーションプラットフォームに正常に同期されました
    • sync_disabled -キャンペーンモジュールが同期を無効にして構成されたため、イベントがスキップされました
    • 失敗 -同期エラー。ツールチップには、発生したエラーの詳細が表示されます
    • unknown_contact -不明なユーザーであるため、イベントが同期されませんでした
    • 該当なし - 適用できない;カスタム、キャンペーントラッキング、またはMarketoMunchkin統合を使用する場合に表示されます

ページの上部にあるドロップダウンリストを使用して、イベントデータをステータス別にフィルタリングすることもできます。

インタラクションイベントの表示

カスタム指標またはイベントがビューイベントにアタッチされている場合、[ インタラクションイベントの表示]リンクが有効になります。たとえば、キャンペーンが有効なプレーヤーでギャラリーのページ内エクスペリエンスを使用していて、エクスペリエンスが次のように構成されている場合リンク、カード、またはその他の種類のインタラクティブ要素、エクスペリエンスはインプレッションイベントとコンバージョンイベントをキャンペーンモジュールに送信します。これらのイベントは、キャンペーンカスタムフィールドを使用してキャンペーンに渡されます。イベントデータは、キャンペーンモジュールに最近のすべての活動。

注:Campaign カスタム指標とイベントを使用する場合も、インタラクションイベントが表示されることがあります。

次のデータが渡されることがあります。

  • イベント 1 -視聴者がインタラクションする機会があった CTA 名のコンマ区切りリスト（印象）
  • イベント 2 - ビューアが操作した CTA 名のカンマ区切りリスト（変換）
  • メトリック 1 -CTA インプレッションの総数
  • メトリック 2 -クリックされたリンク、フォーム送信など、CTA コンバージョンの合計数

イベントを表示するには、ヘッダーの [ インタラクションイベントの表示]リンクをクリックします。

その他のデータ表示オプション

キャンペーンデータは、他の方法で表示/エクスポートできます。

  • ビデオアクティビティ -視聴者のアクティビティをビデオ別に表示するために使用します。詳細については、「ビデオアクティビティの表示」を参照してください。
  • ビューアプロファイル -視聴者によるビデオアクティビティの表示に使用します。詳細は、「ビューアプロファイル情報の表示」を参照してください。
  • データのエクスポート -ビデオビューとリードデータをエクスポートするために使用します。詳細については、を参照してください。キャンペーンモジュールからのキャンペーンデータのエクスポート。
  • オーディエンス API -Audience API を使用して、ビデオビューとリードデータをプログラムで取得できます。詳細については、「概要」を参照してください。オーディエンス API。
