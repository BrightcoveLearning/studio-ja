キャンペーントラッキング統合を使用する場合、ビューアプロファイルは、ビューアを識別するために使用されたデータに基づいて作成されます。ビューアプロファイルを使用すると、特定のビューアの表示データを表示できます。

キャンペーン対応プレーヤーのすべてのビデオビューが追跡されます。ビデオアクティビティを表示するには、をクリックします視聴者プロファイル（非MAP）左側のナビゲーションで。[ビューアプロファイル]ページには、選択した期間のキャンペーン追跡統合からのすべての表示アクティビティのリストが表示されます。カレンダーコントロールを使用して、新しい日付範囲を選択できます。Viewer の値は、ユーザーの識別に使用された値です。通常、これはリードフォームでキャプチャされた電子メールアドレスです。列見出しをクリックすると、データを並べ替えることができます。

注:表示データが表示されるまでに数時間遅れます。

ビューアのリンクをクリックすると、そのビューアのすべての表示アクティビティが表示されます。列見出しをクリックすると、データを並べ替えることができます。

データは、ページの上部にある CSVまたは XLSボタンをクリックしてエクスポートできます。

その他のデータ表示オプション

キャンペーンデータは、他の方法で表示/エクスポートできます。

  • 最近のアクティビティ -過去 24 時間のビデオビューイベントデータを表示するために使用されます。詳細については、を参照してください。キャンペーンモジュールでのプレーヤーイベントの表示。
  • ビデオアクティビティ -視聴者のアクティビティをビデオ別に表示するために使用します。詳細については、「ビデオアクティビティの表示」を参照してください。
  • データのエクスポート -ビデオビューとリードデータをエクスポートするために使用します。詳細については、を参照してください。キャンペーンモジュールからのキャンペーンデータのエクスポート。
  • オーディエンス API -Audience API を使用して、ビデオビューとリードデータをプログラムで取得できます。詳細については、「概要」を参照してください。オーディエンス API。
