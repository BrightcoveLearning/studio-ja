UTM コード (UTM パラメーターとも呼ばれます) は、Web コンテンツの成功を追跡するために URL に追加できる単純なテキストスニペットです。任意の URL に追加できる標準パラメーターは 5 つあります。

  • キャンペーン
  • 出典
  • ミディアム
  • コンテキスト
  • 用語

通常、UTM コードはカスタム URL に追加されます。以下は、URL に追加された UTM コード (太字) の例です。

https://www.brightcove.com/en/blog/2018/08/define-your-video-strategy-3-simple-steps-1?utm_campaign=blogpost&utm_medium=social&utm_source=LinkedIn

UTM コードを使用すると、各キャンペーンのカスタムランディングページを作成しなくても、ブログ投稿、インターネット広告、メールブラストの動作を追跡できます。ブログの投稿、広告、電子メールに別々の UTM コードを作成することで、より多くのトラフィックやコンバージョンを生成するデータを取得できます。

UTM フィールドを操作するためのデータ接続の構成

キャンペーンは、URL引数からUTMフィールドを解析します。UTM データは、マーケティング自動化ソフトウェアから、または Audience API を使用してアクセスできます。必要なキャンペーン構成は、使用されているマーケティング自動化プラットフォームに基づいています。

  • HubSpot
  • Salesforce
  • エロクア
  • Marketto

HubSpot

HubSpot データ接続では、追加のセットアップは必要ありません。UTMフィールドが自動的に表示されます。

セールスフォース

Salesforce データ接続では、[ マッピングタイプ ] を [ 詳細]にして、UTM フィールドを Salesforce の適切なデータオブジェクトにマッピングできます。

Eloqua

Eloquaデータ接続では、追加のセットアップは必要ありません。UTMフィールドが自動的に表示されます。

Marketo

Marketoで作成するカスタムアクティビティに追加できるフィールドが十分でないため、UTMフィールドはMarketo接続ではサポートされていません。
