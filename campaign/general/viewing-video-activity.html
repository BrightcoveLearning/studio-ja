キャンペーントラッキングまたはSSO統合を使用する場合、キャンペーンモジュールを使用すると、ビデオを選択して、そのビデオのアクティビティを表示できます。これにより、特定のビデオを視聴したユーザーを確認できます。

キャンペーン対応プレーヤーのすべてのビデオビューが追跡されます。ビデオアクティビティを表示するには、をクリックしますビデオ（非MAP）左側のナビゲーションで。[アクティビティの表示]ページには、選択した期間にSSOとキャンペーン追跡の統合から表示されたすべてのビデオのリストが表示されます。[接続の選択]コントロールを使用して、特定の接続を選択できます。カレンダーコントロールを使用して、新しい日付範囲を選択できます。列見出しをクリックすると、データを並べ替えることができます。

注意:
  • 表示データが表示されるまでに数時間遅れます。
  • 独立したプラグインとバックエンドシステムを介した分析とキャンペーンの追跡。同じ結果が見られないのは非常に一般的です。キャンペーンは、ブラウザの設定を追跡しないなどにも準拠しています。最も完全な分析については、Analytics モジュールのレポートを参照してください。

動画のリンクをクリックすると、その動画のすべての視聴アクティビティが表示されます。列見出しをクリックすると、データを並べ替えることができます。

データは、ページの上部にある CSVまたは XLSボタンをクリックしてエクスポートできます。

その他のデータ表示オプション

キャンペーンモジュールは、データを表示/エクスポートするいくつかの方法を提供します。

  • 最近のアクティビティ -過去 24 時間のビデオビューイベントデータを表示するために使用されます。詳細については、を参照してください。キャンペーンモジュールでのプレーヤーイベントの表示。
  • ビューアプロファイル -視聴者によるビデオアクティビティの表示に使用します。詳細は、「ビューアプロファイル情報の表示」を参照してください。
  • データのエクスポート -ビデオビューとリードデータをエクスポートするために使用します。詳細については、を参照してください。キャンペーンモジュールからのキャンペーンデータのエクスポート。
  • オーディエンス API -Audience API を使用して、ビデオビューとリードデータをプログラムで取得できます。詳細については、「概要」を参照してください。オーディエンス API。
