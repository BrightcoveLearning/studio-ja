ほとんどのプラットフォームで、電子メールに動画を直接埋込むことはできません。これはブライトコーブの制限ではありません。セキュリティ上の理由により、最新のメールシステムのほとんどすべてが動画プレーヤーの埋め込みコードが取り除かれます。ただし、ティーザーテキストとポスター画像を含むメールを作成することで、コンテンツの認知度とトラフィックを促進することができます。キャンペーンモジュールは、ビデオ、プレーヤーを選択し、電子メールメッセージ内に埋め込むHTMLを生成する機能を提供します。

注:Media モジュールでは、ビデオを電子メールにパブリッシュすることもできます。詳細については、「ビデオを電子メールに公開する」を参照してください。

動画をメールのランディングページに公開するには、次の手順に従います。

  1. キャンペーンモジュールを開きます。
  2. クリックメールに公開左側のナビゲーションで。
  3. ランディングページを選択します。ドロップダウンリストには、シンプルな着陸ページこれは、システムが提供する、スタイルが設定されていないサンプルのランディングページです。このリストには、キャンペーン対応プレーヤーを使用しているギャラリーモジュールを使用して作成された公開済みのランディングページエクスペリエンスも含まれます。ギャラリーのランディングページを使用すると、ランディングページの外観とスタイル設定をより柔軟に行えます。リードフォームを使用して、閲覧者の情報をキャプチャすることもできます。ギャラリーのランディングページの作成の詳細については、「ランディングページエクスペリエンスの作成」を参照してください。
  4. [ ビデオを自動再生する] を選択します。このオプションは、簡易ランディングページを選択した場合にのみ使用できます。
  5. ビデオプレーヤーを選択します。このオプションは、簡易ランディングページを選択した場合にのみ使用できます。ギャラリーのランディングページでは、プレイヤーはエクスペリエンスプロパティの一部として割り当てられます。
  6. ビデオを選択します。検索語句は、アカウント内の動画を検索するために使用できます。
  7. [ イメージサイズ] を設定します。これは、メールの本文に表示される画像です。画像のサイズは、に値を入力することで調整できます幅と身長田畑。プレーヤーの URL がポスター画像の下に表示されます。
  8. 電子メールの送信に使用するアプリケーションを選択します。次のオプションを使用できます。
    • その他
    • エロクア
    • 営業部隊
    • HubSpot
    • パルドット
    • マルケト
    • MailChimp

    埋め込む HTML は、選択したアプリケーションによって異なります。
