Brightcove Social では、ソーシャルメディアアカウントに投稿された動画のパフォーマンスを監視するために使用できるレポートがいくつかあります。

  • ダッシュボード -ソーシャルメディアアカウントの動画パフォーマンスの概要を概説します。詳細については、「ソーシャルアナリティクスの確認」を参照してください。
  • ソーシャルビュー -ソーシャルメディアアカウントで共有された動画のソーシャルビューの概要を提供します。このトピックで説明しています。
  • ソーシャルエンゲージメント -ソーシャルメディアアカウントで共有された動画のソーシャルエンゲージメント（いいね、共有、リツイートなど）の概要を提供します。詳細については、「ソーシャルエンゲージメント分析のレビュー」を参照してください。

レポートの期間を選択することから始めます。ソーシャルビュー分析レポートを表示するには、ページの上部にある [ ソーシャルビュー ] タブをクリックします。

ソーシャルビュー分析レポートは、次のセクションで構成されています。

  • ソーシャルビュー
  • ソーシャルビューの詳細

ソーシャルビュー

[ソーシャルビュー] グラフには、選択した期間におけるソーシャルメディアアカウントに投稿された動画の動画再生数が表示されます。更新日は、Video Cloud がソーシャル分析データを保持している最新の日付です。通常、ソーシャルプラットフォームが API のビューとエンゲージメントデータを更新する前に、24 ～ 48 時間以上の遅延があります。

注:Twitter Insights API はメディアビューを報告しないため、データの表示はソーシャル分析の一部として利用できません。

グラフの右側の凡例を使用して、表示する詳細レベルを選択できます。

  • マイサイトとアプリ -Video Cloud プレーヤーとアプリからの動画再生数を表示します
  • フェイスブック -フェイスブックの動画再生数を表示する
  • YouTube -YouTube の動画再生数を表示する
  • Total -すべてのプラットフォームでビデオ視聴数の合計を表示します。

グラフ内の線の上にマウスポインタを置くと、その日付のデータが表示されます。

ソーシャルビューの詳細

[ソーシャルビュー詳細] セクションは、次の方法によってビデオビューの詳細を表示できます。

  • ビュー (既定)
  • 平均視聴時間
  • 平均% 見た

選択が完了すると、レポートには各動画の詳細データが表示されます。

注:Twitter Insights API はメディアビューを報告しないため、データの表示はソーシャル分析の一部として利用できません。

アナリティクスレポートに表示される YouTube ビューは、 YouTubeアナリティクスの概要レポートから収集されます。YouTube は、このデータがリアルタイムデータであることを保証するものではありません。そのため、このレポートに視聴回数やエンゲージメントが表示されるまでに数日遅れることがあります。データは YouTube アナリティクスに保存されると、ブライトコーブソーシャルアナリティクスの夜間のデータコレクターによって収集されます。

最新のリアルタイム視聴データが必要な場合は、最近公開された上位 25 本の動画について、過去 48 時間の視聴データを見積もったリアルタイムレポートを提供します。YouTube チャンネルのリアルタイムレポートはこちらからご覧いただけます。Brightcove Social Analytics はこのリアルタイムデータを収集しません。これは、実際の動画ではなく、推定視聴回数を提供し、最近の動画にのみ適用されます。

レポートの下部で、表示する行数の選択や他のページへの移動ができます。動画タイトルのリンクをクリックすると、選択した動画のソーシャルビュー分析が表示されます。

よくある質問

1.ソーシャルモジュールを使用して動画を YouTube に公開し、その動画が別のサイトの YouTube プレーヤーに埋め込まれた場合、再生/エンゲージメントデータは引き続き収集され、Social モジュールの分析に渡されますか？

ソーシャルアナリティクスは、クリエイターツールや YouTube のアナリティクスページに表示されるように、YouTube によって報告されたすべてのビューデータを収集します。YouTube 埋め込みプレーヤーのビューがアナリティクスにカウントされる場合、Social モジュールはそれをキャプチャして報告します。埋め込みコード YouTube の [ 共有] > [埋め込み] 機能を使用すると、埋め込みプレーヤーからのビューが YouTube のアナリティクスにカウントされます。ただし、埋め込みコードを編集すると YouTube がビューデータを収集しなくなる可能性があります。特に、埋め込みプレーヤーが動画を自動再生するように設定すると、YouTube はそのプレーヤーからの視聴回数をカウントしない可能性があります。

データのエクスポート

レポートデータのダウンロードは、コンマ区切り形式とExcel形式で可能です。エクスポート機能は、データをから直接保存することに注意してください。 Analytics API電話。エクスポートされたデータはUIデータのようには操作されず、UIで表示されるフィールドのサブセットのみでなく、レポートの基礎となるデータすべてを含んでいます。すべてのデータをAPIの元のフォーマットで保持することで、UIで提供される方法以外でもデータを操作することができます。
