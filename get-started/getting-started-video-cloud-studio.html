Video Cloud Studio にアクセスするには、 https://studio.brightcove.com に移動します。メールアドレスとパスワードでサインインします。ホームページには、次の機能があります。

  • システム通知の表示
  • ビデオクラウドの設定を管理する
  • アカウントを選択し、ユーザープロファイルを表示する
  • ビデオクラウドモジュールにアクセスする
  • VideoCloudダッシュボードを表示する

いつでも最大5つのセッションを持つことができます。セッションは、同じメールアドレスでログインするたびに開始されます。6 番目のセッションが開始されると、最も古いセッションは無効になります。Video Cloud Studio セッションのタイムアウトは 24 時間です。

システム通知を表示する

Video Cloud Studioヘッダーにベルアイコンが表示されます 利用可能なシステム通知がある場合。このアイコンをクリックすると、システム通知のリストが表示されます。

Video Cloud Studioヘッダーに警告アイコンが表示されます システムに問題がある場合。アイコンをクリックすると、ブライトコーブのシステムステータスページが開きます。

設定を管理する

Video Cloud Studioの設定メニューには、管理者メニュー。Video Cloud の管理メニューでは、次の項目にアクセスできます。

  • アカウント情報 -現在の Video Cloud アカウントに関する情報を表示します
  • ユーザーの管理 -Video Cloud Studio の使用を許可されたユーザーを管理します。
  • アカウント設定 -一般的なアカウント設定の構成
  • アップロード設定 -アップロードモジュールをカスタマイズするためのオプションを提供します
  • プロファイルの取り込み -Video Cloud アカウントで取り込みプロファイルを管理する
  • 動画フィールド -標準の動画項目を必須としてマークし、アカウントのカスタムメタデータフィールドを表示、作成、変更する
  • 言語設定 -追加のメタデータ言語のサポートを構成します
  • 再生権 -再生権を設定します
  • メディア共有 -他のアカウントで動画を共有する
  • サーバー側の広告設定 -サーバー側の広告設定を構成します
  • プレーヤーの設定 -Player Management and Delivery SystemAPIへのアクセスに基本的なユーザー名とパスワード認証を使用するオプションを提供します
  • 統合 -Zoom ミーティングをブライトコーブライブに統合する
  • ライブ設定 -ライブモジュールで使用される設定を管理します。
  • API 認証 -API 認証情報の作成と管理
  • 請求 -アカウントの請求情報の表示
注:[ADMIN] メニューにアクセスするには、アカウント管理者である必要があります。

アカウントを選択する

複数の Video Cloud アカウントにアクセスできる場合は、Video Cloud アカウントセレクターを使用して、アクセスする Video Cloud アカウントを選択できます。アカウントを切り替えるには、Video Cloud Studio の右上隅にある自分の名前-アカウントをクリックします。現在のアカウントは、アカウント名の前に付けられた点で示されます。

[ マイプロファイル]リンクをクリックして、ユーザー情報とパスワードを更新します。詳細は、「プロファイル情報の管理」を参照してください。

モジュールへのアクセス

Video Cloud Studio メニューでは、Studio モジュールにアクセスできます。[ ホーム] をクリックして、メニューにアクセスします。

表示されるモジュールは、お客様のVideo Cloud契約およびユーザープロファイルによって異なります。

  • アップロード -Video Cloud アカウントに動画をアップロードする
  • メディア -ビデオのプロパティの設定、フォルダーを使用したビデオの整理、ビデオの公開
  • アナリティクス -動画とプレイヤーのパフォーマンスに関するアナリティクスを表示
  • プレーヤー -プレーヤーの作成とスタイル設定を行い、プレーヤーの公開コードを生成する
  • ライブ -ライブイベントの作成とストリーミング
  • クラウド再生 -動画用のライブリニアチャンネルを作成する
  • ギャラリー -ビデオコンテンツ用のビデオポータルの作成
  • ソーシャル -ソーシャルメディアアカウントにビデオを配布する
  • 運動 -ビデオ分析をVideoCloudからマーケティングオートメーションプラットフォームに接続します

ダッシュボードを表示する

Video Cloud Studio のホームページに Video Cloud ダッシュボードが表示されます。ダッシュボードには一連の「カード」が含まれており、アカウント内の動画に関する高いレベルの指標が表示されます。以下のカードが利用可能です。

  • ビデオビュー -選択した期間の 1 日あたりのビデオビュー数を表示します。「ビデオビューの詳細を見る」リンクをクリックすると、パフォーマンスレポートが開き、ビデオのパフォーマンスを確認できます。
  • アカウントの概要 -アカウント内の動画とプレーヤーの合計数、および最近作成および更新された動画の数が表示されます。数字をクリックすると、Media または Players モジュールが開きます。
  • デバイスタイプ -選択した期間でビデオを視聴するために使用されるデバイスの種類が表示されます。[ デバイスタイプ別にビデオビューを表示]リンクをクリックすると、パフォーマンスレポートが開き、デバイスの種類別にビデオのパフォーマンスを表示できます。
  • 新品は何ですか？ -Video Cloud 製品のアップデートを表示します。アップデートの完全なリストについては、 Video Cloud Studio リリースノートを参照してください。
  • トップトレンド動画 -選択した期間の視聴回数が多い動画を表示します。カードタイトルのリンクをクリックすると、パフォーマンスレポートが開き、ビデオのパフォーマンスを確認できます。
  • [リージョンの表示] -選択した期間のビューアの場所を表示します。[ 国別ビデオビューを表示] をクリックすると、パフォーマンスレポートが開き、国別にビデオビューを表示できます。

ローカライズ

Video Cloud Studio の言語は、プロフィール内の言語を変更することで変更できます。次の言語がサポートされています。

  • 英語
  • スペイン語 (スペイン)
  • 日本人
  • フランス語 (フランス)
  • 中国語 (繁体字)
  • 韓国語

詳細は、「プロファイル情報の管理」を参照してください。
