このクイックスタートのすべてのトピックを確認すると、次のことができるようになります。

  • Video Cloud スタジオ内を移動する
  • Video Cloud に動画をアップロードする
  • ビデオプロパティの編集
  • 単一のビデオプレーヤーを作成してスタイル設定する
  • 動画を Web に公開する
  • プレイリストを作成する
  • プレイリストプレーヤーを作成する
  • プレイリストを Web に公開する

対象者

Video Cloud を試している、もしくは初めて使用される方。

前提条件

  • Video Cloud アカウント- こちらでトライアルアカウントにサインアップできます。
  • 使用できるサンプルビデオがあるか、いくつかのサンプルビデオをダウンロードできるはずです。

概要

この概要の各トピックでは、Video Cloud を使用した動画公開ワークフローの特定の部分について説明します。各トピックには概要説明のビデオと、それに続く実践的な手順があります。すべてのトレーニングビデオを見るだけで、実践的な演習を行わない場合は、オンデマンドトレーニングコース Video Cloud Basicsをご覧ください。

  • 動画をアップロードする
  • 基本的なビデオプロパティの編集
  • プレイヤーを作成する
  • プレーヤーをスタイリングする
  • 1 つのビデオを Web に公開する
  • 手動プレイリストを作成する
  • スマートプレイリストを作成する
  • プレイリストプレーヤーを作成する
  • プレイリストプレーヤーを公開する

このクイックスタートでは、Video Cloud Analytics モジュールは対象外です。Video Cloud分析の詳細については、Video Cloud分析入門オンデマンドトレーニングコースをご覧ください。

次は何をすべきですか？

  • Video Cloud の詳しい掘り下げについては、Video Cloud 入門オンデマンドトレーニングコースをご覧ください。
  • 利用可能なビデオ分析の種類については、Video Cloud Analytics の概要オンデマンドトレーニングコースをご覧ください。
  • Galleryを使用してビデオポータルまたはページ内エクスペリエンスを公開する方法については、 ギャラリーを使用したポータルエクスペリエンスの構築またはギャラリーを使用したページ内エクスペリエンスの構築をご覧ください。オンデマンドトレーニング。
