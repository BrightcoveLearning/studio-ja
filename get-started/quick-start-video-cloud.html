このクイックスタートのすべてのトピックを確認すると、次のことができるようになります。

  • ビデオクラウドスタジオ内を移動する
  • Video Cloud に動画をアップロードする
  • ビデオプロパティの編集
  • 単一のビデオプレーヤーを作成してスタイル設定する
  • 動画を Web に公開する
  • プレイリストを作成する
  • プレイリストプレーヤーを作成する
  • プレイリストを Web に公開する

聴衆

評価している、または Video Cloud を初めて使用している人。

前提条件

  • Video Cloud アカウント- こちらでトライアルアカウントにサインアップできます。
  • 使用できるサンプルビデオがあるか、いくつかのサンプルビデオをダウンロードできるはずです。

アウトライン

この概要の各トピックでは、Video Cloud を使用したビデオパブリッシングワークフローの特定の部分について説明します。各トピックには概要ビデオがあり、その後に実行できる実践的な手順が続きます。すべてのトレーニングビデオを見たいだけで、ハンズオン演習を完了しない場合は、 Video Cloud Basicsオンデマンドトレーニングコースをご覧ください。

  • 動画をアップロードする
  • 基本的なビデオプロパティの編集
  • プレイヤーを作成する
  • プレーヤーをスタイリングする
  • 1 つのビデオを Web に公開する
  • 手動プレイリストを作成する
  • スマートプレイリストを作成する
  • プレイリストプレーヤーを作成する
  • プレイリストプレーヤーを公開する

このクイックスタートでは、Video Cloud Analytics モジュールは対象外です。ビデオクラウド分析の詳細については、ビデオクラウド分析入門オンデマンドトレーニングコースをご覧ください。
