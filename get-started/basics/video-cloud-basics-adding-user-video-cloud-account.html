ステップ

  1. ビデオクラウドスタジオにログインします。
  2. ナビゲーションヘッダーで、[ 管理者 ]、[ ユーザー管理] の順にクリックします。「管理者」メニューは、管理者ロールを持つユーザーのみが使用できます。
  3. ユーザーの電子メール、名、姓を入力します。
  4. ロールをユーザーに割り当てます。
    • 管理者 -ユーザーはにアクセスできます管理者メニュー
    • Standard - ユーザーは管理者メニューにアクセスできません
  5. ユーザーがアクセスできるモジュールを選択します。モジュールをオフにすると、ユーザーがアクセスできるStudioのセクションが制限されます。
  6. [ユーザーを追加]をクリックします。

ユーザーは、アカウントに追加されたことを示す電子メールメッセージを受信します。ページの本文には、アカウントへのアクセス権を持つすべてのユーザーが一覧表示されます。
