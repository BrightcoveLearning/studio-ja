ステップ

  1. メディアモジュールを開きます。
  2. サイド・ナビゲーション・パネルで、「フォルダの作成」リンクをクリックします。
  3. フォルダの名前を入力し、 Enter キーを押します。
  4. フォルダーにビデオを追加するには、1 つ以上のビデオを選択します。
  5. [詳細] > [フォルダに移動] をクリックします。
  6. フォルダをクリックし、 [移動] をクリックします。

すべてのフォルダが左側のナビゲーションに表示されます。フォルダ名の後の数字は、フォルダ内の動画の数です。

フォルダーからビデオを削除するには、フォルダーをクリックし、1 つまたは複数のビデオを選択し、[ 動画を削除] > [フォルダーから動画を削除] をクリックします。
