ステップ

  1. Video Cloud Studio で、[ アップロード ] をクリックして [ アップロード]モジュールを開きます。
  2. [ ビデオの参照] をクリックし、アップロードするファイルを参照して選択するか、ファイルシステムからアップロードモジュールにファイルをドラッグアンドドロップします。
  3. ファイルの処理が開始されます。すべての動画のステータスが [ 処理完了] になるまで、[アップロードモジュール] ページにとどまります (ページを更新しないでください)。
  4. 「ビデオの詳細を表示して編集」をクリックして、ビデオを表示し、メディアモジュールでビデオプロパティを編集します。
