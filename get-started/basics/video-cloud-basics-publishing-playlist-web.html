前提条件

  • Video Cloud アカウントに動画をアップロードしました
  • 手動プレイリストまたはスマートプレイリストを作成している

ステップ

  1. メディアモジュールを開きます。
  2. 公開するプレイリストを選択:
    • サイド・ナビゲーションにプレイリストが表示されたら、そのプレイリストをクリックし、「プレイリストの公開」をクリックします。
    • 左側のナビゲーションメニューで [ すべてのプレイリスト ] をクリックし、プレイリストを選択し、[ 公開と埋め込み] > [Web に公開] の順にクリックします。
  3. [プレーヤーの選択]ドロップダウンリストで、使用するプレーヤーを選択します。
  4. クリックURLプレビュープレーヤーの下で、新しいブラウザタブで開きます。プレイリストが表示され、ビデオを再生できることを確認します。
  5. 埋め込みコードをコピーして Web ページに貼り付けます。
