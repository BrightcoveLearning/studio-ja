ステップ

  1. Video Cloud Studio で、[ プレーヤー ] をクリックして [ プレーヤー]モジュールを開きます。
  2. [新規プレーヤー]をクリックします。
  3. 「名前」と「短い説明」を入力します。
  4. [保存]をクリックします。

プレーヤーのプロパティページを表示するには、プレイヤー名をクリックします。
