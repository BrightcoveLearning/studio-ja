ステップ

  1. Brightcove システムステータスページにアクセスするには、Video Cloud Studio 内からサポート >システムステータスをクリックするか、 status.brightcove.com にアクセスしてください。

インシデントステータスは、次のいずれかになります。

  • 緑 -動作可能
  • 青 -通知-システムは現在動作中ですが、遅延または断続的な停止の影響を受ける
  • 黄色: 部分的なサービス中断
  • 赤 -フルサービスの中断
