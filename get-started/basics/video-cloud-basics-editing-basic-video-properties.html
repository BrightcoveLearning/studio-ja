前提条件

  • Video Cloud アカウントに動画をアップロードしました

ステップ

  1. Video Cloud Studio で、[ メディア ] をクリックして [ メディア]モジュールを開きます。
  2. ビデオタイトルをクリックして、ビデオのプロパティページを開きます。
  3. ビデオプロパティはセクションに分かれています。ビデオのプロパティを編集するには、[ 編集 ] をクリックするか、セクションをダブルクリックします。
  4. [保存] をクリックして、変更を保存します。
