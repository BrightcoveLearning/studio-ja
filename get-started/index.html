入門編

Video Cloud Studio を使用すると、デスクトップ、タブレット、モバイルデバイスのビデオパブリッシングワークフローを管理できます。動画をアップロード、管理、公開するための使いやすいインターフェイスを提供します。このページでは、Video Cloud Studio を使用してビデオワークフローを管理し、理解し、成功に導くための主要な概念について説明します。詳細については、次の概念を参照してください。

こちらで無料トライアルアカウントにサインアップしてください。
