トライアルの制限事項

Video Cloud トライアルアカウントには、次の制限があります。

  • アカウントにアップロードできる動画は最大 5 つです。アカウントがトライアルモードで5本以上の動画をアップロードできるようにする必要がある場合は、当社の営業チームに連絡して、アップロードのための無制限のアクセスを提供します。ブライトコーブのパートナー統合に取り組んでいる場合は、パートナーチームに直接お問い合わせください。
  • 取り込みプロファイルにアクセスできません
  • メディア共有は利用できません
  • 画像のアップロードはギャラリーモジュールでは利用できません
  • API 管理にアクセスできません

はじめに

Video Cloud の使用を開始するために、次のリソースを利用することをお勧めします。

ビデオクラウドの基本

ビデオクラウドの基本は、Video Cloud の使用を開始するのに役立つ一連の短いビデオです。ビデオは、ビデオをアップロードして公開するプロセスを順を追って説明します。

ビデオ公開ワークフロードキュメント

Video Cloud Studio公開ワークフロードキュメントには、ビデオをアップロード、管理、公開するための一連の手順が用意されています。このドキュメントは、「Video Cloud 入門」トレーニングの一部として使用されます。このトレーニングは、基本トレーニングよりも深いものです。

トレーニング

ブライトコーブは、オンライントレーニングとオンデマンドトレーニングの両方を提供しています。

トレーニングビデオ

Video Cloud のさまざまな機能を紹介するトレーニングビデオが利用可能で、こちらでご覧いただけます。

サポート

Video Cloud トライアルで質問や問題がある場合は、ブライトコーブサポートにリクエストを送信してください。
