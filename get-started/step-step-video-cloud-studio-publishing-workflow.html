Video Cloud に動画をアップロードする

先に進む前に、サンプルビデオを使用またはダウンロードできるサンプル動画をいくつか用意しておく必要があります。

Video Cloud アカウントにログインする

  1. ブラウザウィンドウで https://studio.brightcove.com に移動します。
  2. 資格情報を入力し、[ サインイン] をクリックします。

ローカル動画を Video Cloud にアップロードする

  1. [Upload]をクリックして、Uploadモジュールを開きます。
  2. アップロードモジュールで、取り込みプロファイルを選択します。
  3. [ 動画の参照] をクリックし、提供された動画またはすべてのサンプル動画を参照して選択します。
  4. このページには、すべての動画とアップロードの進行状況が表示されます。

    アップロードが処理状態にある間は、ブラウザウィンドウを閉じたり、ブラウザウィンドウから離したりしないでください。そうすると、アップロードが不完全になる可能性があります。

    アップロードが完了すると、「処理完了」ステータスが表示されます。これは、処理が完了し、アップロードモジュールから安全に移動できることを示します。

    注:クリックすることが可能です編集ファイル名の横にあるボタンをクリックして、ビデオメタデータの一部を編集します。ビデオメタデータの追加については、次のクイックスタートトピックで説明します。

Video Cloudにリモート動画をアップロードする

  1. Upload モジュールで、[URLを入力する]をクリックします。
  2. AmazonS3アカウントでホストされているファイルへの次のURLを入力します。
    http://learning-services-media.brightcove.com/vc-quick-start/oystercatcher.mp4
  3. [ ビデオの処理] をクリックして、アップロードを開始します。

（オプション）モバイルデバイスを使用して動画をアップロードする

  1. モバイルデバイスを使用して、 https://studio.brightcove.com で Video Cloud Studio を開きます。
  2. アップロードモジュールを開きます。
  3. 取り込みプロファイルを選択します。
  4. [ ビデオの参照] をクリックします。デバイスのカメラロールアプリケーションが開きます。
  5. アップロードする動画を選択します。アップロードが開始され、アップロードの進行状況が表示されます。

プレイヤーの作成とスタイリング

このタスクでは、プレイヤーを作成し、スタイルを設定します。

新しいプレーヤーを作成する

  1. Video Cloud Studio で、[ プレーヤー ] をクリックして [プレーヤー] モジュールを開きます。
  2. [ 新しいプレーヤー] をクリックします。
  3. Custom Player名前を入力し、[ 保存] をクリックします。

プレーヤーのスタイリング

  1. [ カスタムプレーヤー]リンクをクリックして、プレーヤーのプロパティを開きます。
  2. プレイヤータイプがシングルビデオであることを確認します。
  3. 左側のナビゲーションメニューで [ スタイリング ] をクリックします。
  4. [ タイトルと説明を表示] をオンにします。
  5. [ 再生ボタンの形状]と [ 再生ボタンの位置] を選択します。
  6. Play Buttonカラーセレクターをクリックし、色を赤の色に設定します。
  7. 色の変化がプレビュープレーヤーに反映されていることを確認します。
  8. [ パブリッシュと埋め込み] > [変更の公開] をクリックして、変更を公開します。

動画を Web に公開する

このタスクでは、ビデオを Web に公開します。

1 つの動画を公開する

  1. Video Cloud Studio で、[ メディア ] をクリックして [メディア] モジュールを開きます。
  2. Brown Pelicanビデオの横にあるチェックボックスをクリックし、[ 公開と埋め込み] > [Web に公開] の順にクリックします。
    注:その他の公開オプションについては、対応するドキュメントを参照してください。
    • Eメールランディングページ - ビデオを電子メールに公開する
    • ソーシャルネットワーク - ソーシャルメディアプラットフォームへのビデオの公開
    • ギャラリーのページ内エクスペリエンス - ページはめ込みエクスペリエンスへの動画の公開
  3. [公開]ダイアログが開くことを確認します。
  4. [プレーヤーを選択]ドロップダウンリストで、[ カスタムプレーヤー] を選択します。このプレーヤーは前のセクションで作成されました。
    注:Brightcoveのデフォルトプレーヤーは、すべての Video Cloud アカウントにデフォルトで作成されます。
  5. クリックURLプレビュープレーヤーの下で、新しいブラウザタブで開きます。ビデオが表示され、再生できることを確認します。
    注:短縮 URLを生成するには、[短縮] をクリックします。
  6. （オプション）URLをモバイルデバイスに電子メールで送信します。リンクをクリックして、ビデオがモバイルデバイスで再生されることを確認します。

プレイリストを作成する

このタスクでは、手動およびスマートプレイリストを作成します。

手動プレイリストを作成する

  1. Video Cloud Studio で、[ メディア ] をクリックして [メディア] モジュールを開きます。
  2. サイド・ナビゲーション・パネルで、「プレイリストを作成」リンクをクリックします。[新しいプレイリスト] ダイアログが開きます。
  3. プレイリストの種類を [ 手動] に設定します。
  4. プレイリスト名をに設定しますFavorites。
  5. サイドバーにプレイリストを表示するには、[サイドバーにプレイリストを表示]を選択します。
  6. [保存]をクリックします。

マニュアルプレイリストに動画を追加する

  1. クリックすべてのビデオアカウント内のすべてのビデオを表示するには、サイドナビゲーションパネルで。
  2. 再生リストに追加する動画の横にあるチェックボックスをクリックします。
  3. [ その他] > [プレイリストに追加] をクリックします。
  4. [ お気に入り]プレイリストを選択し、[ 追加] をクリックします。
  5. サイド・ナビゲーション・パネルの「お気に入り」プレイリストをクリックし、動画がプレイリストに追加されたことを確認します。
  6. 動画の順序を並べ替えるには、行をクリックし、リストの新しい位置までドラッグします。

スマートプレイリストを作成する

  1. サイド・ナビゲーション・パネルで、「プレイリストを作成」リンクをクリックします。
  2. プレイリストの種類を [ スマート] に設定します。
  3. プレイリスト名をに設定しますBirds。
  4. サイドバーにプレイリストを表示をオンにして、プレイリストをサイドバーに表示します。
  5. をセットするタグフィールドに1つ以上が含まれていますと入力しますbirdsタグ用。
  6. [保存]をクリックします。
  7. クリック鳥サイドナビゲーションパネルでプレイリストを表示し、鳥の動画がプレイリストに追加されていることを確認します。

プレイリストの公開

このタスクでは、新しいプレーヤーを作成し、プレイリストをサポートするようにプレーヤーの設定を編集し、プレイリストを公開します。

プレイリストをサポートするようプレーヤーの設定を編集する

注:レスポンシブサイズ設定オプションを選択すると、標準 (iframe) コードがうまく機能し、プレイヤーとプレイリストの両方が応答します。詳細設定（ページ内埋め込み）コードを使用すると、プレーヤーは応答しますが、プレイリストは応答しません。
  1. Video Cloud Studioにログインし、[Players]をクリックして、Playersモジュールを開きます。
  2. [ 新しいプレーヤー] をクリックします。プレイヤーの名前を付けPlaylist Player、[ 保存] をクリックします。
  3. プレイリストプレーヤーを見つけて、プレイヤー名をクリックしてプレーヤーのプロパティを開きます。
  4. を選択プレイヤータイプのプレイリスト。
  5. アスペクト比を 16:9 に設定します。
  6. プレーヤーの寸法を [ 幅]に設定します640。
  7. [ 保存] をクリックします。
  8. 左側のナビゲーションメニューで [ スタイリング ] をクリックします。
  9. [ プレイリストタイプ]の [ プレイリスト (垂直)] をクリックします。
  10. 左側のナビゲーションメニューで [ 再生 ] をクリックします。
  11. [ 選択したときにビデオを再生する] をオンにします。
  12. 連続再生モードをチェックします。
  13. パブリッシュと埋め込み >変更を公開をクリックして、変更をプレイヤーに公開します。

プレイリストの公開

  1. [ メディア ] をクリックして、メディアモジュールを開きます。
  2. クリックすべてのプレイリストサイドナビゲーションパネルで、すべてのプレイリストを表示します。
  3. Birdsプレイリストの横にあるチェックボックスをクリックします。
  4. [ パブリッシュと埋め込み] > [Web にパブリッシュ] をクリックします。
  5. プレイリストプレーヤーを選択します。
  6. URLリンクをクリックします。プレーヤーは新しいブラウザタブで開きます。プレイリストプレーヤーが表示され、プレイリスト内のビデオを再生できることを確認します。
  7. モバイルデバイスに電子メールでプレーヤーURLを送信します（オプション）。リンクをクリックして、プレイリストプレーヤーが表示され、プレイリスト内のビデオを再生できることを確認します。
