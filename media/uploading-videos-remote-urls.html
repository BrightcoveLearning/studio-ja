アップロードモジュールは、動的な取り込みプロセスを使用して動画コンテンツを取得およびトランスコードします。アップロードするファイルを選択すると、Video Cloud はプルベースのプロセスを使用してストレージロケーションからコンテンツをプルし、Zencoder トランスコーディングサービスを使用してコンテンツをトランスコードします。

アップロードモジュールは、ビデオのアップロードのさまざまな方法をサポートしています。

  • ドラッグ・アンド・ドロップ -この方法では、ファイルをローカル・ファイル・システムから「モジュールのアップロード」ページにドラッグできます
  • ファイル選択ダイアログ -この方法では、ファイル選択ダイアログが開き、アップロードするファイルを参照して選択します
  • リモート URL -このメソッドでは、URL を入力してソースファイルの場所を指定できます。

このトピックでは、一般的なクラウドファイルホスティングサービスのリモート URL を取得する方法について説明します。リモート URL アップロード方法を使用すると、組織外のユーザーによって編集/処理されたサイズの大きいファイルには便利です。大きなファイルをローカルマシンにコピーするのではなく、ファイルへのリモート URL を提供でき、Video Cloud はその場所からファイルをプルできます。いずれの場合も、ファイルのURLが公開されていることを確認する必要があります。そうしないと、アップロードが失敗します。

注:Upload モジュールの使用方法の詳細については、アップロードモジュールを使用したビデオのアップロードを参照してください。
注:動画をアップロードする前に、適切な取り込みプロファイルが選択されていることを確認してください。

グーグルドライブ

Google ドライブに保存された動画の URL を取得するには、次の手順に従います。

  1. Google ドライブアカウントにログインします。
  2. アップロードしたい動画を見つけて右クリックします。
  3. [ 共有可能なリンクを取得] をクリックします。
  4. 権限については、リンクを持つすべてのユーザーがファイルにアクセスできるように、必ずオプションを選択してください。権限が Restricted に設定されている場合、アップロードは失敗します。
    注:企業のGoogleアカウントをお持ちの場合は、共有オプションのみが制限されているか、ドメインに共有されている可能性があります。リンクを持つ誰にでもファイルを共有できない場合、Video Cloud はファイルにアクセスできず、アップロードは失敗します。
  5. [ リンクのコピー] をクリックします。残念ながら、これはビデオファイルへの直接リンクを生成しません。私たちは、共有リンクからビデオファイルへの直接リンクを生成するために、別のサイトを使用することができます。
  6. https://sites.google.com/site/gdocs2direct/home に移動します。Google ドライブファイルへの直接リンクを生成するサイト以外のサイトも利用できます。
  7. 共有URLを貼り付けます。
  8. [ ダイレクトリンクを作成] をクリックします。

[出力リンク]フィールドに返されるURLは、アップロードモジュールでビデオをアップロードするために使用するURLです。

注:動画ファイルへの直接リンクが生成されないため、YouTube URL は使用できません。
注:Googleドライブのサイズ警告が表示されている場合（通常はファイルの場合）、アップロードは機能しません>100MB）。URL を変更した後でも、サイズエラーページにリダイレクトされ、アップロードは失敗します。

ドロップボックス

Dropbox に保存された動画の URL を取得するには、次の手順に従います。

  1. Dropbox アカウントにログインします。
  2. アップロードする動画を見つけて、[ 共有]リンクをクリックします。
  3. [ リンクの作成] をクリックして、ファイルへのリンクを作成します。
  4. [ リンクのコピー] をクリックして、リンクをファイルにコピーします。

ファイルの URL がで終わる場合があることに注意してください?dl=0。ブラウザがリンクの内容をダウンロードするように強制するには、URL のその部分をに変更する必要があります?dl=1。

ボックス

Boxに保存されているビデオのURLを取得するには、次の手順に従います。

注:ファイルへの直接リンクを取得できるのはBox Premium機能で、無料の個人アカウントでは利用できません。
  1. Boxアカウントにログインします。
  2. アップロードする動画を見つけて、[ 共有]リンクをクリックします。
  3. [ 共有リンクの作成]スライダをクリックして、ファイルへのリンクを作成します。これにより、ファイルへのプレビューリンクが作成されます。
  4. [ リンク設定] をクリックします。ファイルに直接ダウンロードリンクを割り当てる必要があります。
  5. [ コピー ] をクリックして、リンクをクリップボードにコピーします。

その他のサービス

注: OneDrive URL はサポートされていません。

URL が公開されている URL である限り、他のクラウドプラットフォーム（Amazon S3）でホストされている動画ファイルの URL の使用に問題はありません。

注:Amazon S3からビデオファイルを取り込む場合、バケットポリシーを作成して、すべての人が利用できるわけではないビデオにVideoCloudからアクセスできるようにすることができます。詳細については、開発者ドキュメント「S3 での動的インジェストの使用」を参照してください。
