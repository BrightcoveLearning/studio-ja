Video Cloud アカウントの動画には、ポスターとサムネイル画像が関連付けられています。これらの画像を使用すると、視聴者はプレイリストやコンテンツのリストでどの動画を見たいかを選択できます。ポスターとサムネイル画像は、以下のエリアで見ることができます。

  • Media モジュールでは、サムネイル画像はビデオの隣に表示されます。ポスター画像は、ビデオのプロパティページのプレビュープレーヤーに表示されます。
  • プレイリストを公開すると、サムネイル画像が表示されます
  • ビデオ再生の前後にビデオプレーヤーの再生エリアにポスター画像が表示されます

Video Cloud にアップロードされた動画の場合、中間ポイントでスナップショットを作成することで、ポスター画像とサムネイル画像が自動的に作成されます。Video Cloud は、最高品質の再生可能な MP4 レンディションから画像をキャプチャしようとします。MP4レンディションがない場合は、次にHLSレンディションを試し、次にDASHレンディションを試します。

注:HLS レンディション（MP4 レンディションのみ）を含まないカスタム取り込みプロファイルを使用してビデオを取り込む場合、トランスコーディング処理中にポスターとサムネイルがキャプチャされません。このトピックで説明するプロセスを使用して、イメージをキャプチャできます。

クライアント側の画像キャプチャは、ブラウザが再生できる最大のレンディションのフレームを取得し、デフォルトの取り込みプロファイルで指定されたポスターとサムネイルのサイズを使用して、VideoCloud取り込みプロセスに送信します。これらのプロファイル寸法がキャプチャできたフレームよりも大きい場合、画像のサイズ変更や拡大/縮小は行われません。サイズが小さい場合、イメージは縮小されます。ポスターおよびサムネイル画像の品質を向上させる方法については、「ポスターおよびサムネイル画像の品質を上げる」を参照してください。

ポスターやサムネイルとして使用する他の画像をアップロードすることもできます。

必要条件

イメージキャプチャ機能には、現在、次の要件があります。

  • Chrome、Safari（バージョン10以降）、FirefoxとEdgeブラウザがサポートされている
  • ビデオレンディションは、CORSヘッダーを使用して提供する必要があります（これは、BrightcoveのほとんどのハウスCDNでデフォルトで有効になっている必要があります）。 BYO CDNを使用している場合は、 CORSヘッダーを有効にする画像キャプチャが機能するようにサーバー上で
  • Chrome と Firefox の最新バージョンを使用している場合、画像キャプチャはモバイルデバイス上で動作するはずです
  • 画像キャプチャは、DRM を使用してセキュリティで保護されている動画ではサポートされていません
  • 制限付きトライアルアカウントでは、画像キャプチャはサポートされていません。この制限を削除するには、ブライトコーブにお問い合わせください。

画像キャプチャが使用できない場合は、通常、画像キャプチャボタンは無効になります。このルールの例外は、ビデオがレガシープロファイルを使用して取り込まれた場合です。この場合、ボタンは有効になりますが、画像キャプチャは機能しません。回避策は、ダイナミック配信プロファイルまたはコンテキスト対応エンコーディングプロファイルを使用してビデオを再トランスコードまたは置換し、イメージキャプチャを使用できるようにすることです。

新しいポスターとサムネイル画像をキャプチャする手順

新しいポスターまたはサムネイル画像をキャプチャするには、次の手順に従います。

  1. ナビゲーションヘッダーの [ メディア ] をクリックして、Media モジュールを開きます。
  2. 編集する動画のリンクをクリックします。
  3. ページ上部にあるプレビュープレーヤーを使用して、ビデオを再生し、ポスター/サムネイル画像として保存したいフレームを一時停止します。プレーヤーの進行状況バーをスクラブして、別のフレームにジャンプすることもできます。
  4. [ キャプチャ] をクリックします。
  5. ページの上部にあるプレビュープレーヤーには、現在のビデオフレームが表示されます。チェックボックスを使用して、新しいポスターまたはサムネイル画像を取り込むかどうかを選択します。
  6. [ 保存 ] をクリックして、新しいイメージを保存します。処理が完了し、イメージが Media モジュールに表示されるまでに数分かかる場合があります。

制限事項

  • 取り込みシステムでは、同じビデオで同時ジョブを実行できません。したがって、ビデオの取り込みまたは再変換中に画像をキャプチャしようとすると、画像のキャプチャは失敗します。
