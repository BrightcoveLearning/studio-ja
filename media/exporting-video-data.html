メディアモジュールは、すべてのビデオまたはビデオのサブセットのビデオメタデータをエクスポートする機能を提供します。この機能を使用するときは、次の点に注意してください。

  • ブライトコーブでは、一度にエクスポートできる動画を 15K 未満にすることをお勧めします。15Kを超える動画をエクスポートする必要がある場合は、検索機能を使用して動画のサブセットを返し、一度に複数の「バッチ」をエクスポートします。
  • レンディションとイメージの URL は固定されません。ブライトコーブは、メディアアセットの保存を随時再設定します。この場合、特定のアセットの URL が変更されます。これらのアセットへのハードコードされた URL に依存している場合、リンクがある時点で破損する可能性があります。メディアへのリンクが壊れないようにする最善の方法は、 CMS API を使用して実行時に Video Cloudからメディアを取得することです。
  • Excel で開いた場合、韓国語や日本語など 2 バイト文字が正しく表示されない。これは、生成された.csvファイルがUTF-8によってエンコードされ、Excelが適切にサポートしていないことが原因です。これを修正するには、.csvファイルをANSIで再エンコードする必要があります。その後、Excelで正しく表示されます。

ビデオメタデータを書き出すには、次の手順に従います。

  1. Media モジュールを開きます。
  2. 特定の動画のデータをエクスポートする場合は、動画の横にあるチェックボックスをオンにします。
  3. クリック書き出すページの上部にあります。
  4. エクスポートするビデオを選択します。
    • このリスト内のすべての動画 -アカウント内のすべての動画の動画メタデータをエクスポートします
    • x 選択した動画 -選択した動画のみの動画メタデータを書き出します。
  5. ビデオレンディションの URL を含めるには、[ビデオレンディションの現在の URL を含める] をオンにします。多数のビデオを選択してこのオプションをチェックすると、エクスポートファイルの生成に必要な時間が長くなります。
    注:このオプションを選択すると、最大 1000 のビデオのビデオメタデータをエクスポートできます。
  6. [ データのエクスポート] をクリックします。データのエクスポートが開始されます。
  7. エクスポートファイルの準備ができたら、[ Download File ] をクリックして、ビデオデータファイルをローカルコンピュータにダウンロードします。

.csvファイルには、VideoCloudアカウント番号で名前が付けられます。サードパーティのスプレッドシートプログラムを使用して、書き出したビデオデータを表示できます。
