通常は、Uploadモジュールを使用して、動画をVideo Cloudにアップロードします。これらのファイルは、ブライトコーブによって管理されます。リモートアセットでは、ご自身のシステムまたはサードパーティのシステムで動画ファイルを維持管理し、動画ファイルのURLをVideo Cloudに提供します。

注:リモートアセットビデオを作成する機能は、Video CloudEnterpriseパブリッシャーが利用できます。Video Cloud アカウントのアップグレードに興味がある場合は、Brightcove にお問い合わせください。

リモートアセット動画は、Video Cloud CMS APIを使用して作成することもできます。詳細については、開発者向けドキュメントを参照してください。を使用して作成されたリモートアセットビデオCMS APIその後、メディアモジュールを使用して管理できます。

リモートアセットにアクセスする

すべてのリモートアセットファイルは、認証やアクセス制御の必要がなく、パブリック URL で利用可能である必要があります。リモートアセットが CDN によってホストされている場合、リモートアセットの URL は、オリジンサーバー上の URL ではなく CDN 内のアセットの URL である必要があります。

リモートアセットを作成するビデオ

リモートアセットビデオを作成するには、次の手順に従います。

  1. メディアモジュールを開きます。
  2. クリックリモートビデオを追加するサイドナビゲーションパネルで。[リモートビデオの追加]ダイアログが開きます。
  3. ビデオ名を入力します。ビデオ名は、Mediaモジュールとプレーヤーにも表示されます。
  4. を選択ビデオタイプ、住むまたオンデマンド。
  5. にとってオンデマンドビデオタイプ、入力間隔 hh: mm: ss形式。
  6. レンディションごとに、レンディションの URL とフォーマットを入力します。[ レンディションを追加] をクリックします。
    注:プロトコル相対 URL がサポートされています。
  7. [保存]をクリックして、リモートアセットを保存します。ページが更新されると、リモートアセットビデオがメディアモジュールに一覧表示されます。

リモートアセットビデオにはポスターやサムネイル画像はありませんが、画像をアップロードすることはできます。ポスターおよびサムネイル画像のアップロードについては、ポスターおよびサムネイル画像のアップロードを参照してください。

リモートアセットのビデオの編集

リモートアセットビデオを編集するには、次の手順に従います。

  1. メディアモジュールを開きます。
  2. リモートアセットのビデオタイトルをクリックして、ビデオのプロパティを開きます。
  3. [ ビデオファイル-リモートアセット]セクションを探し、[ 編集] をクリックします。
  4. [ ソース]行をクリックして選択します。ソースURLをクリックすると、別のブラウザタブで開きます。
  5. を更新しますURLと間隔必要に応じて。[ 保存 ] をクリックして変更を保存します。

制約事項

  • 画像キャプチャ機能を使用して、リモートアセットビデオのビデオ静止画とサムネイル画像を作成することはできませんが、リモートアセットビデオのビデオ静止画とサムネイル画像をアップロードすることはできます
  • で作成されたリモートアセットビデオ基本アカウントの作成後に編集することはできません
  • SSAIはリモートアセットではサポートされていません
  • DRM は、プレーヤーをカスタマイズして DRM ライセンスサーバーと通信できる場合にのみサポートされます。これは、プラグインを通じて Brightcove Player で可能です。他のプレイヤーについては、プレーヤーのドキュメントを参照する必要があります。
  • アナリティクス:ブライトコーブアナリティクスシステムは、動画の再生時間などのシステム生成動画メタデータに依存します。リモートアセットの分析データは、このメタデータを提供できない限り、リモートアセットの分析データのリッチは低くなります。
  • 再生保証:Video Cloud に取り込まれたすべての動画は、ブライトコーブプレーヤーで再生できます。これは、ほぼすべてのリモートアセットに当てはまりますが、動画が再生されないエッジケースもあります。例えば、DASHとHLSには、まだサポートしていないいくつかの高度な機能があります。BrightCove 以外のプレーヤーの場合は、そのプレーヤーが動画形式を完全にサポートしていることを確認する必要があります。
  • コンピューターの時刻が実際の時刻よりも早く設定されている場合、視聴者はリモートアセットビデオを再生できない場合があります
  • ライブリモートアセットビデオにキューポイントを追加することはできません
