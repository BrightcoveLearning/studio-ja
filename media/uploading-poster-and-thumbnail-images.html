Video Cloud アカウントの動画には、ポスターとサムネイル画像が関連付けられています。これらの画像を使用すると、視聴者はプレイリストやコンテンツのリストでどの動画を見たいかを選択できます。ポスターとサムネイル画像は、以下のエリアで見ることができます。

  • Media モジュールでは、サムネイル画像はビデオの隣に表示されます。ポスター画像は、ビデオのプロパティページのプレビュープレーヤーに表示されます。
  • プレイリストを公開すると、サムネイル画像が表示されます
  • ポスター画像は、プレーヤーがロードされ、ビデオのメタデータが使用可能になった後、ビデオプレーヤーの再生領域に表示されます。視聴者が再生ボタンを押すまでポスターが表示されます

Video Cloud にアップロードされた動画の場合、中間ポイントでソースビデオからスナップショットを取得することにより、ポスターおよびサムネイル画像が自動的に作成されます。ポスターとサムネイルのサイズは、ビデオのアップロード時に使用される取り込みプロファイルで指定されている寸法に基づきます。キャプチャされたデフォルトの画像が気に入らない場合は、他の画像をアップロードしたり、新しい画像をキャプチャしてポスターやサムネイルとして使用できます。

ノート:
  • 最近アップロードした動画の画像をアップロードする場合は、動画の処理が完全に完了するまで待つ必要があります。そうしないと、画像のアップロードが失敗します。
  • アップロードされた画像の処理は非同期プロセスであるため、発生したエラーはすぐにはわかりません。30秒後にページを更新して、画像が正常にアップロードおよび処理されたことを確認する必要があります。

カスタムポスターとサムネイル画像をアップロードする場合は、次のことをおすすめします。

  • ポスター画像 -推奨アスペクト比 16:9、最小幅は640px
  • サムネイル画像 -推奨アスペクト比16: 9、最小幅160px
    ノート: アップロードされた画像は sRGB カラー形式を使用する必要があります。

新しいポスターとサムネイル画像をアップロードすると、画像のサイズは、アカウントのデフォルトの取り込みプロファイルで指定された画像のサイズに基づきます。ポスターおよびサムネイル画像の品質を向上させる方法については、「ポスターおよびサムネイル画像の品質を上げる」を参照してください。

新しいポスターとサムネイル画像をアップロードする手順

新しいポスターまたはサムネイル画像をアップロードするには、次の手順に従います。

  1. ナビゲーションヘッダーの [ メディア ] をクリックして、Media モジュールを開きます。
  2. 編集する動画のリンクをクリックします。
  3. [ イメージ]セクションを見つけます。
  4. ポスター/サムネイル画像にカーソルを合わせ、アップロードボタンをクリックします。
  5. 新しい画像を参照して選択します。処理が完了し、イメージが Media モジュールに表示されるまでに数分かかる場合があります。
    注: .pngおよび.jpg画像がサポートされています。

制約事項

  • 取り込みシステムでは、同じビデオで同時ジョブを実行できません。したがって、ビデオの取り込みまたは再変換中に画像をアップロードしようとすると、画像のアップロードは失敗します。
  • 従来のプロファイルを使用してビデオが取り込まれた場合、アップロードボタンは有効になりますが、画像のアップロードは失敗します。回避策は次のとおりです。再トランスコードまた交換 DynamicDeliveryまたはContextAwareEncodingプロファイルを使用したビデオ。
