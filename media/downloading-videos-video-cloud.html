Video Cloudアカウントから動画をダウンロードするにはいくつかの方法があります。API を使用して、Media または Social モジュールを使用してビデオをダウンロードできます。また、ダウンロードリンクをプレーヤーに追加して、視聴者がコンテンツをダウンロードできます。

Mediaモジュールを使用してビデオをダウンロードする

メディアモジュールを使用して、ビデオのレンディションをダウンロードできます。メディアモジュールのビデオ名の横にあるクラウドアイコンが灰色の場合（）、ビデオは従来の取り込みプロファイルを使用して取り込まれ、MP4とHLSの両方のレンディションをメディアモジュールにダウンロードできます。緑の雲のアイコン（）は、ビデオがDynamic Deliveryを使用して取り込まれたことを示し、MP4レンディションを作成する取り込みプロファイルが選択されている場合は、MP4レンディションをダウンロードできます。

ノート:
  • ビデオプロパティページの[ビデオファイル]セクションには、使用された取り込みプロファイルのタイプも示されます。
  • 1080p レンディションをダウンロードできるようにするには、カスタムインジェストプロファイルを作成し、次のレンディションのうち少なくとも 1 つが含まれていることを確認する必要があります。
    • progressive3500
    • progressive4000

メディアモジュールを使用してビデオをダウンロードするには、次の手順に従います。

  1. メディアモジュールを開きます。
  2. ダウンロードするビデオのリンクをクリックします。
  3. [ ビデオファイル ] セクションを見つけます。
  4. 使用されたトランスコーディングの種類を確認します。
    • Dynamic Delivery (緑の雲アイコン、以下の例)-レンディションをダウンロードするには、[ ディメンション]列のリンクを右クリックします。クリック可能なダウンロードリンクは、MP4 の場合のみ表示されます。レンディションが作成されました
    • レガシー (灰色の雲アイコン)-レンディションをダウンロードするには、[ フォーマット ] 列のリンクを右クリックします。
注:Dynamic Deliveryによって生成されたレンディションの存続時間（TTL）は6時間です。生成されたリンクは 6 時間有効です。

レンディションと画像のURLは固定されていないことを理解することが重要です。ブライトコーブは、メディアアセットの保存を随時再設定します。この場合、特定のアセットの URL が変更されます。これらのアセットへのハードコードされた URL に依存している場合、リンクがある時点で破損する可能性があります。メディアへのリンクが壊れないようにする最善の方法は、CMS API を使用して実行時に Video Cloud から取得することです。

Social モジュールを使用したビデオのダウンロード

Social モジュールは、ビデオ URL をコピーしたり、ビデオソースファイルをダウンロードしたりする機能を提供します。ダウンロード時に、最高品質の MP4 レンディションがダウンロードされます。

Social モジュールを使用してビデオをダウンロードするには、次の手順に従います。

  1. ブライトコーブソーシャルを開きます。
  2. クリックビデオ左側のナビゲーションで。Video Cloud アカウントの動画の一覧が表示されます。
  3. ダウンロードするビデオをクリックし、[ ダウンロード] をクリックします。
    注:MP4 レンディションがない場合、[ ダウンロード ] リンクは無効になります。
    注:DRM が有効になっている動画はダウンロードできません（動画のサムネイルにロックが表示されます）。DRM 対応ビデオを選択すると、[ ダウンロード]ボタンが無効になります。
  4. 動画URLをコピーするか、またはリンクをクリックして、ブラウザーで動画を開きます。ソースファイルをローカルに保存するには、動画を右クリックします。
  5. [完了]をクリックします。

CMSAPIを使用したビデオプロパティへのアクセス

NS CMS API VideoCloudアカウント内の任意のビデオのビデオURLを含むメタデータを取得するために使用できます。CMS APIを使用するには、開発スキルが必要です。CMS API を使用してビデオプロパティにアクセスする方法については、「CMS API の使用:ビデオデータデベロッパードキュメントを取得します。

Brightcove LearningServicesもサンプルを作成しましたCMS APIダウンロードリンクを取得するために使用できるアプリ: CMS API サンプル:ダウンロードリンク。

ビデオファイルのダウンロードリンクの作成

開発スキルがある場合は、プレーヤーのコントロールバーにダウンロードリンクを配置して、視聴者にコンテンツをダウンロードするオプションを提供することができます。

ダウンロードリンクをプレーヤーオーバーレイまたはプレーヤーの下に表示することもできます。サンプルコードなど、これを行う方法の詳細については、 Brightcove Player のサンプルを参照してください。ビデオプラグインをダウンロードします。
