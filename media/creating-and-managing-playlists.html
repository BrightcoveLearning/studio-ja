プレイリストは、プレーヤーで再生するために特定の順序でグループ化されているビデオのコレクションです。Media モジュールを使用すると、プレイリストの作成や再生リストへのビデオの追加が簡単に行えます。プレイリストを作成したら、プレイリストをプレイヤーに公開できます。プレイリストの公開の詳細については、「メディアモジュールを使用したビデオとプレイリストの公開」を参照してください。

プレイリストの詳細については、プレイリストを効果的に使用するためのベストプラクティスやヒントなど、プレイリストの概要を参照してください。

プレイリストの種類

プレイリストには、次の 2 つのタイプがあります。

  • 手動 -手動プレイリストでは、特定の動画を再生リストに割り当てて、手動で並べ替えます。
  • スマート -スマートプレイリストでは、ビデオタグやその他のビデオプロパティに基づいて、再生リストのビデオのリストが自動的に作成されます。スマートプレイリストは動的で、メディアライブラリ内の動画とプレイリストを定義するルールに基づいて自動的に更新されます。

プレイリストに追加できる動画は最大100本（手動とスマートの両方）。作成できるプレイリストの数に制限はありません。プレーヤーのプレイリストに最初に読み込まれる動画は、再生リストのタイプによって決まります。

  • 手動プレイリスト -動画は、手動再生リストのビデオの順序に基づいて再生リストに読み込まれます。
  • スマートプレイリスト -再生順序プレイリストの設定に基づいて、ビデオがプレイリストにロードされます。

プログラムで追加の動画を再生リストに取り出す方法については、 Playback APIページングプレーヤーのサンプルを参照してください。

プレイリストを作成する

プレイリストを作成するには、Media モジュールを開き、次の手順に従います。

  1. サイド・ナビゲーション・パネルで、「プレイリストを作成」をクリックします。新規プレイリストダイアログが開きます。
    注:サイドナビゲーションボタンをクリックすると、サイドナビゲーションパネルを表示または非表示にできます 。
  2. プレイリストのタイプを[手動]または[スマート]から選択します。
  3. 次のガイドラインに従って、プレイリスト名を入力します。
    • 名前は100文字に制限されています
    • 不正な文字:/
  4. （任意）。参照 ID と説明を入力します。
  5. サイドバーにプレイリストを表示するには、[サイドバーにプレイリストを表示]を選択します。
    注:サイドバーには最大 100 個のプレイリストを表示できます。
  6. [保存]をクリックします。

スマートプレイリストの操作

スマートプレイリストを作成する場合は、以下の設定も使用できます。

  • タグ - タグを使用して、プレイリストに表示する動画を定義します。[ タグ ] ドロップダウンには、次のオプションがあります。
    • 1 つ以上を含む -1 つ以上のタグがリストされている動画を含めます。
    • すべてを含む -プレイリスト内のすべての動画にリストされているすべてのタグを持っていることを要求します

    タグテキストフィールドに、プレイリストを定義するために使用する 1 つ以上のタグ（カンマで区切って）を入力します。

  • 再生順序 -再生リストのビデオを再生する順序を制御します。次のオプションを使用できます。
    再生順序 説明
    有効日 (古い順) 動画がアクティブ化された日付に基づいて、最も古い動画が最初になります
    有効日 (新しい順) 動画が有効化された日付に基づいて、最新の動画が最初に表示されます
    ビデオ名 (A-Z) 動画名のアルファベット順
    総再生数 合計ビデオ再生数に基づきます
    最後の週再生 最後の週の総動画再生数に基づきます[1]
    開始日（古い順） 動画の開始予定日に基づいて、最も古い動画が最初になります。動画に開始日が指定されていない場合、開始日順に並べられた再生リストには表示されません。注:ビデオスケジューリングは、Video Cloud エンタープライズアカウントでのみ使用できます。
    開始日 (新しい順) 動画の開始予定日に基づいて、最新の動画が最初に表示されます。動画に開始日が指定されていない場合、開始日順に並べられた再生リストには表示されません。注:ビデオスケジューリングは、Video Cloud エンタープライズアカウントでのみ使用できます。

    注

    • [1] 動画メタデータに保持されている最後の週の再生データは最大2日経過している可能性があるため、Analyticsモジュールで過去7日間の動画再生を確認した場合に表示される動画と一致しない可能性があることに注意してください。
  • 動画数の制限 - プレイリストに入れる動画の最大数を入力します。

動画を手動プレイリストに追加する

手動プレイリストに動画を追加するには、次の手順に従います。

  1. プレイリストに追加する必要がある動画の横にあるチェックボックスをオンにします。
  2. ページの上部にある [ その他] > [プレイリストに追加] をクリックします。
  3. 動画を追加するプレイリストを選択します。
  4. [ 追加] をクリックします。手動プレイリストに追加した動画は、再生リストの先頭に追加されます。

プレイリストの動画の順序を変更する

再生リストのビデオの順序を変更するには、操作している再生リストのタイプに応じた手順に従います。

手動再生リストのビデオの順序を変更する

  1. 次のいずれかの方法を使用して、プレイリストを開きます。
    • サイド・ナビゲーション・パネルのプレイリスト名をクリックします
    • クリックすべてのプレイリストサイドナビゲーションパネルで、プレイリストを見つけて、プレイリスト名のリンクをクリックします。
  2. プレイリストの動画の一覧が表示されます。
  3. ビデオをクリックし、新しい場所にドラッグします。暗い線は新しい位置を示します。
注:[ 再生順序 ] フィールドを編集して、ビデオの順序を変更することもできます。
ドラッグアンドドロップによる並べ替えはいいえ IE11またはEdgeで動作しますが、の番号を編集することでアイテムを並べ替えることができます注文を再生分野。

スマートプレイリスト内の動画の順序を変更する

  1. サイドバーにプレイリストが表示されている場合は、プレイリストをクリックし、プレイリストの横にある下向き矢印をクリックし、[ 設定の編集] をクリックします。
  2. サイドバーにプレイリストが表示されない場合は、サイド・ナビゲーションの「すべてのプレイリスト」の見出しをクリックし、プレイリストを見つけてプレイリスト名のリンクをクリックし、「設定の編集」をクリックします。
  3. プレイリストの新しい再生順序を選択します。
  4. [ 保存] をクリックします。

スマートプレイリストは、プレーヤーが利用可能なビデオ情報から読み込まれるときに動的に作成されるため、順序には常に現在の状態が反映されている必要があります。ただし、プレーヤーは常に、最大20分間キャッシュされたデータを使用するPlaybackAPIを介して情報を取得します。プレイリストは更新に時間がかかる場合があり、 CMS API表示されます。

プレイリストの設定を編集する

手動プレイリストまたはスマートプレイリストの設定を編集するには、次の手順に従います。

  1. サイドバーにプレイリストが表示されている場合は、プレイリストをクリックし、プレイリストの横にある下向き矢印をクリックし、[ 設定の編集] をクリックします。
  2. サイドバーにプレイリストが表示されない場合は、サイド・ナビゲーションの「すべてのプレイリスト」の見出しをクリックし、プレイリストを見つけてプレイリスト名のリンクをクリックし、「設定の編集」をクリックします。
  3. プレイリストの設定を編集し、[保存]をクリックします。
あなたがするかもしれないことに注意してくださいいいえ手動プレイリストをスマートプレイリストに、またはその逆に変換します。

サイド・ナビゲーション・パネルにプレイリストが表示されている場合は、プレイリストをクリックし、プレイリストの横にある下向き矢印をクリックし、「設定の編集」をクリックします。

プレイリストの削除

プレイリストを削除するには、次の手順に従います。

  1. [ すべてのプレイリスト ] をクリックして、すべてのプレイリストのリストを表示します。
  2. 削除するプレイリストの隣のチェックボックスをチェックします。
  3. 削除ボタンをクリックします（）。
  4. [ プレイリストの削除] をクリックして、削除を確定します。

サイド・ナビゲーション・パネルにプレイリストが表示されている場合は、プレイリストをクリックし、プレイリストの横にある下向き矢印をクリックして、「削除」をクリックします。

手動プレイリストから動画を削除する

手動再生リストから動画を削除するには、次の手順に従います。

  1. 次のいずれかの方法を使用して、手動プレイリストを開きます。
    • サイド・ナビゲーション・パネルのプレイリスト名をクリックします
    • クリックすべてのプレイリストサイドナビゲーションパネルで、プレイリストを見つけて、プレイリスト名のリンクをクリックします。
  2. プレイリストの動画の一覧が表示されます。
  3. 削除する動画の横にあるチェックボックスをクリックします。
  4. [ プレイリストから削除] をクリックします。

ビデオのプロパティを編集して、手動再生リストからビデオを削除することもできます。[ 分類 ] セクションで、[ 手動プレイリスト]セクションからプレイリストを削除します。
