ビデオの空き時間プロパティを使用して、ビデオを視聴できる日付と時刻を制御できます。地域フィルタリングを使用して、ビデオを見ることができる国を制御できます。

ノート:
  • 日付のスケジュールに使用されるデータ形式は、米国の形式を使用します。MM/DD/YYYY。
  • ビデオの空き状況プロパティは、エンタープライズアカウントとプロフェッショナルアカウントでのみ使用できます。
  • ビデオのスケジュールは、ビデオがアクティブであるか、非アクティブになるかには影響しません。スケジュールは、アクティブな動画にのみ影響します。
  • ジオフィルタリングとスケジューリングのプロパティは、 CMS APIあなたが作成またアップデートビデオ。
  • 一部のサイト運営者は、米国だけでなく米軍基地にも動画をジオフィルタリングする機能を要求しています。これをサポートするために、米軍の選択肢があります。軍事基地を識別する 100% 信頼できる方法はありませんが、.mil ドメインに解決する IP が検出され、米国内にない場合、実際の原産国ではなく米軍基地から来るものとして識別されます。米国にある.mil アドレスは、引き続き「米国」として識別されます。

動画の公開設定プロパティを表示、編集するには、Mediaモジュールを開き、動画タイトルをクリックして、[公開設定]セクションを見つけます。

アクセス制限プロパティを編集するには、[編集]ボタンをクリックするか、そのセクションをダブルクリックします。ビデオ可用性は、次の条件に基づいて設定できます。

  • 日付と時刻
  • ジオフィルタリング

日付と時刻の設定

動画には、開始日/時刻と終了日/時刻を設定できます。日付/時刻は、ビデオがいつ表示可能になるかを指定します。

デフォルトでは、ビデオはすぐに利用可能で、終了日はありません。開始日/終了日時を設定するには、次の手順に従います。

  1. 日付/時刻フィールドの横にあるラジオボタンをクリックします。
  2. カレンダーアイコンをクリックし、カレンダーコントロールを使用して日付を選択します。
  3. 時計アイコンをクリックします（）時間を設定します。
    注:タイムゾーンは、コンピュータまたはデバイスに設定されているタイムゾーンに基づきます。
  4. カレンダーコントロールに戻るには、カレンダーアイコンをクリックします（）。
  5. [保存]をクリックします。

地理フィルタリングの設定

地域フィルタリングを使用して、ビデオを再生できる国とできない国を制御できます。

注意:
  • プレイヤーの地情報フィルタリング機能は、エンタープライズアカウントとプロフェッショナルアカウントで有料で利用できる機能です。地域フィルタリングを含めるようにアカウントをアップグレードする場合は、ブライトコーブにお問い合わせください。
  • 地域フィルタリングはプレイヤーにも適用できます。Video Cloud は、プレーヤーと動画地理フィルターの両方のポリシーを適用します。いずれかの国で除外された場合、その動画はそこで利用できなくなります。

ビデオの geo フィルタリングプロパティを設定するには:

  1. [ 次の国からの閲覧を許可する]または [ 次の国からの表示を禁止する] を選択します。
  2. 適切な国名を入力します。入力すると、国リストがフィルタリングされます。
  3. [保存]をクリックします。
