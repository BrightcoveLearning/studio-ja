ビデオがVideoCloudにアップロードされると、途中でソースビデオからスナップショットを取得することにより、ポスターとサムネイルの画像が自動的に作成されます。ポスターとサムネイルのサイズは、ビデオのアップロード時に使用される取り込みプロファイルで指定されている寸法に基づきます。

ポスターとサムネイル画像は、以下のエリアで見ることができます。

  • Media モジュールでは、サムネイル画像はビデオの隣に表示されます。ポスター画像は、ビデオのプロパティページのプレビュープレーヤーに表示されます。
  • プレイリストを公開すると、サムネイル画像が表示されます
  • ポスター画像は、プレーヤーがロードされた後、ビデオプレーヤーの再生領域に表示されます。ポスターは、視聴者が再生ボタンを押すまで表示されます
注:キャプチャされたデフォルトの画像が気に入らない場合は、新しい画像をアップロードするか、新しい画像をキャプチャして、ポスターとサムネイルとして使用できます。

公開されているプレーヤーのサイズによっては、ポスター画像がプレーヤーに合わせて拡大される場合があります。画像を拡大縮小すると、ぼやけたり、歪んだように見えることがあります。ポスター画像を大きいサイズにしておくと、画像を拡大縮小する必要がありません。

画像サイズの確認

ポスター画像のサイズを確認する方法はいくつかあります。

  • Media モジュールを使用してイメージサイズを確認します。
  • 取り込みプロファイルで画像サイズを確認する

Media モジュールでのイメージサイズの確認

Media モジュールでポスターとサムネイル画像のサイズを確認するには、次の手順に従います。

  1. Media モジュールを開きます。
  2. ビデオをクリックして、ビデオのプロパティを開きます。
  3. の中に画像セクションで、ポスター/サムネイル画像にカーソルを合わせます。
  4. ビューアイコンをクリックします（）新しいブラウザタブで画像を開きます。

画像の正確な寸法を表示できる多くのツールのいずれかを使用することもできます。あなたのプレーヤーのどれかがポスターのサイズより大きい場合、画像は拡大されます。

取り込みプロファイル画像サイズの確認

Video Cloud Studio には、ビデオのアップロード時に使用された取り込みプロファイルは表示されません。ただし、取り込みプロファイルのイメージサイズを確認することはできます。

  1. Studio ヘッダーで、[ 管理者]リンクをクリックします。管理者メニューにアクセスするには、管理者ロールを持つ Video Cloud Studio ユーザーである必要があります。
  2. [ プロファイルの取り込み] をクリックします。
  3. プロファイル名をクリックします。プロファイルエディタにプロファイル定義が表示されます。 にとって動的配信とコンテキストアウェアプロファイル、画像サイズはにリストされています画像プロファイル定義のセクション。次の例は、動的配信プロファイルバージョンからのものです。
  4. ポスターと画像のサイズを確認してください。あなたのプレーヤーのどれかがポスターのサイズより大きい場合、画像は拡大されます。
注:標準の取り込みプロファイルの画像サイズもドキュメントに記載されています。 動的配信およびコンテキスト対応エンコーディングの標準インジェストプロファイル。

カスタム取り込みプロファイルの作成

イメージサイズを変更するには、カスタム取り込みプロファイルを作成し、サイズを適切な値に設定する必要があります。カスタム取り込みプロファイルの作成については、「取り込みプロファイルの管理」を参照してください。

注:画像サイズを大きくすると、管理コンテンツと帯域幅の使用量が増加しますが、動画の再生レートが極端に低い場合、未再生動画用に読み込まれた画像の帯域幅が実際の動画再生よりも多く消費される場合を除き、無視できます。
注:取り込みプロファイルを作成および編集できる機能は、Enterprise アカウントと Professional アカウントを持つパブリッシャーのみが利用できます。

新しい取り込みプロファイル（または複数のプロファイル）が作成されたら、ビデオをアップロードするときにそのプロファイルを使用する必要があります。

注:取り込みプロファイルの標準セットは編集または削除できません。

自動サイズ変更

Studio または Dynamic Ingest API 経由でカスタムポスターやサムネイル画像を追加すると、デフォルトでは、Video Cloud は、アカウントのデフォルトの取り込みプロファイルで定義されている画像サイズに合わせてこれらのサイズを変更します。

既存の動画はどうですか？

既存の動画の画像サイズを修正する最良の方法は何ですか？次の 2 つのオプションがあります。

  • 既存の動画を再トランスコード
  • 新しいポスター画像をアップロードする

既存の動画を再トランスコード

既存の動画を再変換すると、新しいポスター画像とサムネイル画像が生成されます。ビデオは、より大きな画像サイズの取り込みプロファイルを使用して再変換する必要があります。既存の動画の再トランスコードについて詳しくは、Media モジュールを使用したビデオの再トランスコードを参照してください。

新しいポスター画像をアップロードする

Media モジュールを使用して、新しい適切なサイズのポスター画像をアップロードできます。新しいポスター画像がアップロードされると、そのアカウントのデフォルトの取り込みプロファイルを使用して画像がトランスコードされます。デフォルトの取り込みプロファイルのビデオサムネイル画像のサイズが適切であることを確認する必要があります。

デフォルトの取り込みプロファイルを表示するには、取り込みプロファイルのリストを表示するときに、 インジケータ。
