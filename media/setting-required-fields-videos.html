多くのパブリッシャは、コンテンツを追跡し収益化するための不可欠のツールとして、動画メタデータ（名前、タグ、参照ID、カスタムフィールドなど）を使用します。ビデオのメタデータを有効にするには、使用している各メタデータフィールドが実際に存在していることを確認する必要があります。メタデータ保証機能を使用すると、ビジネス目的で重要な動画メタデータフィールドを指定し、動画をアクティブにする前に、新しい動画に必要なメタデータフィールドに入力するようユーザーに要求できます。

メタデータフィールドを必要に応じて指定する

必須のビデオメタデータフィールドは、Video Cloud アカウント設定の一部として設定されます。標準メタデータフィールドとカスタムメタデータフィールドの両方を必須としてマークできます。必須フィールドを構成するには、次の手順に従います。

  1. Studio ヘッダーの [ 管理者]リンクをクリックします。
  2. [ ビデオフィールド]リンクをクリックします。
  3. 必要に応じて、設定するフィールド名をクリックします。
  4. [ このフィールドを必須にする]チェックボックスをオンにします。
  5. [ フィールドを保存] をクリックします。

このフィールドを必須にしたくない場合は、いつでも任意のフィールドの [ このフィールドを必須にする]チェックボックスをオフにすることができます。それ以降、Video Cloud はそのフィールドに入力されているかどうかをチェックしなくなります。

必須フィールドにできるメタデータフィールドは何ですか？

ビデオ名フィールドは常に必須です。任意のカスタムメタデータフィールドを必須フィールドにすることができます。さらに、次の標準メタデータフィールドも必須です。

  • 広告
  • ジオフィルタリングオン
  • 短い説明
  • 長い説明
  • 参照ID
  • 関連リンクテキスト
  • 関連リンク URL
  • スケジューリング開始日
  • スケジュール終了日
  • タグ

必要なメタデータがない動画

メディアライブラリ内のビデオに必要なメタデータがすべて含まれていない場合は、次のようになります。

  • すでにアクティブになっている場合は、アクティブで再生可能なままです。つまり、アカウントに必要なメタデータフィールドを指定しても、必要なメタデータを更新するまで、既存のビデオライブラリが突然再生不能になることはありません。
  • すでにアクティブで、Media モジュールで編集している場合は、編集中のセクションに必要なメタデータをすべて指定しないと、編集内容を保存できません。
  • 非アクティブの場合は、必要なメタデータをすべて指定しないと、アクティブ化できません。

必要なメタデータなしで動画をアップロードする

デフォルトでは、Uploadモジュールを使用して動画をアップロードすると、動画のステータスは次のように設定されます。アクティブ必須フィールドが欠落していない限り。必須フィールドがない場合、メッセージが表示され、必須フィールドが赤色で強調表示されます。必須フィールドが欠落している動画は引き続きアップロードできますが、すべての必須フィールドに値が設定されるまで非アクティブのままになります。

ビデオの編集

ビデオがアクティブでない場合、必要なメタデータをすべて指定することなく、Media モジュールでメタデータを自由に編集および保存できます。また、必要なメタデータがすべて入力されるまで、ビデオをアクティブ化することはできません。

動画がすでにアクティブな場合、編集中のセクションの変更を保存するには、そのセクションの必須メタデータをすべて入力するか、または動画を非アクティブ化する必要があります。

動画をアクティブ化する

Media モジュールでビデオをアクティブにするには、必要なすべてのメタデータが必要です。必須メタデータが欠けている動画をアクティブにしようとすると、エラーが表示されます。

下のステータスバーには、動画をアクティブ化できるかどうかが表示されます。[Activate] ボタンは、ビデオに必要なすべてのメタデータを指定するまで無効のままになります。

ワークフローのヒント

アクティブでない動画をすべて見つけたいですか？メディアモジュールでは、すべての動画をステータスで並べ替えることができます。[ ステータス]列のヘッダーをクリックするだけです。すると、非アクティブな動画を編集し、不足する必須メタデータを確認することができます。

制約事項

必須フィールドは、Media モジュールでのみ適用されます。Media モジュールでアクティブでないビデオは、必須のメタデータフィールドがすべて入力されていない限り、アクティブ化できません。ただし、必須メタデータフィールドのない動画は、次の方法でアクティブになる可能性があります。

  • 動画はアカウントで作成され、アカウントの必須フィールドを指定する前に有効化されています
  • 動画は動的取り込み API を使用してアップロードされました

さらに、Video Cloud はタイプを検証しますが、必要なメタデータの値は検証しません。入力した値が [アカウント設定] のフィールドにリストされている正しいタイプ (テキスト、URL、日付など) であれば、ユーザーは選択した任意の値を入力できます。
