アップロードモジュールを使用してビデオをアップロードする場合、または DI API を使用してダイナミックインジェストリクエストを作成する場合は、Video Cloud の取り込みプロセスによって作成されるレンディションの数とタイプを決定するインジェストプロファイルを指定する必要があります。このトピックでは、各レガシー取り込みプロファイルの詳細について説明します。動的配信プロファイルの詳細は、「動的配信の標準取り込みプロファイル」を参照してください。

注:低品質（低ビットレート）のソースコンテンツをアップロードし、標準プロファイルのいずれかを選択すると、Video Cloud はソースと同等または低い品質のレンディションのみを生成します。

独自のカスタム取り込みプロファイルを作成することもできます。詳細は、「取り込みプロファイルの管理」を参照してください。

注意:
  • このページは API のライブデータを使用して構築されています。完全にロードされるまでに数分かかる場合があります。
  • 標準プロファイルは編集できません。変更したバージョンが必要な場合は、カスタムプロファイルを作成する必要があります。
