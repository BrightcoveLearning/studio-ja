プロフィールはいくつありますか？

最大 200 個のカスタムプロファイルを定義できます。ただし、取り込みジョブのプロファイルを選択する際の混乱を避けるため、通常は必要な数だけ作成する必要があります。

プロフィールはどのように管理するのですか？

Studio の [ 取り込みプロファイル管理] ページを使用して、取り込みプロファイルを管理できます。このインターフェイスでは、次のようなさまざまなオプションを使用できます。

  • デフォルトプロファイルの設定
  • 新しいプロファイルの作成
  • 既存のプロファイルからのクローン作成
  • カスタムプロファイルの編集
  • 不要なプロファイルを削除する

取り込みプロファイルへのプログラムによるアクセスが必要な場合は、を使用してカスタムプロファイルを作成、読み取り、更新、および削除できます。プロファイルの取り込みRESTAPI。これには OAuth 認証情報を設定する必要があることに注意してください。

プロフィールを更新するにはどうすればよいですか？

上記のセクションを参照してください。

プロファイルにはレンディションの数はいくつありますか？

動的配信プロファイルには、最大 37 のレンディションを使用できます。カスタム動的配信プロファイルの場合、現在使用可能な事前定義されたレンディションのみを選択できます。

min_renditionsカスタムコンテキスト対応エンコーディングプロファイルを作成している場合、の最小値は 1 max_renditionsで、最大値は 15 です。

不要なレンディションは、管理ストレージコストを増大させるため、公開するビデオの種類やターゲットとする視聴者やデバイスでは、再生されにくいレンディションを作成しないでください。

アカウントの標準プロファイルは、一般的な視聴者の帯域幅と最もよく使用されるデバイスで、優れた視聴体験を提供するように設計されています。

カスタムプロファイルの命名

カスタムプロファイル名には、スペースや特殊文字を含めることはできません。英数字を使用し、スペースはダッシュ (-) またはアンダースコア (_) に置き換えます。

ユーザーエラーを回避するには、プロファイルを何に使うべきかを動画をアップロードしたユーザーに明確に伝える方法で、プロフィールに名前を付けてください。

デフォルトとして設定すべきプロフィールは何ですか？

標準プロファイルまたはカスタムプロファイルは、StudioまたはIngest Profile APIを使用してデフォルトとして設定できます。デフォルトは、ユーザーが特定のプロファイルを指定しない場合（アップロードモジュールまたは動的取り込み API を使用して）場合にのみ使用されます。

ユーザーが特定のプロファイルを指定するのを忘れた場合、既定のプロファイルを設定することをお勧めします。
