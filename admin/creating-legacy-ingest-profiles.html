[Ingest Profiles] 管理ページを使用して、新しいレガシー取り込みプロファイルを作成できます。取り込みプロファイルは JSON を使用して定義されます。新しいレガシープロファイルを作成する場合は、「レガシートランスコーディングの標準取り込みプロファイル」ドキュメントを確認し、必要なものと同様のプロファイルを見つけると便利です。その後、Studioで簡単にプロファイルを複製し、変更を加えて編集できます。

注:DRM を使用する取り込みプロファイルの作成については、 MPEG-DASHビデオ開発者ドキュメントを参照してください。

注:各プロファイルでは、最大 25 のレンディションを作成できます。 ポスターとサムネイル画像は、最大 25 にカウントされます。 

新しいレガシー取り込みプロファイルを作成するには、次の手順に従います。

  1. Studio ヘッダーの [ 管理者]リンクをクリックします。
  2. [ プロファイルの取り込み]リンクをクリックします。
  3. 複製するレガシープロファイルを探します。プロファイル名をクリックして、プロファイルエディタを開きます。
  4. [ プロファイルの複製] をクリックします。プロファイルエディタが開き、新しいプロファイルが表示されます。
  5. プロフィールの上部までスクロールします。name要素を編集して、プロファイルに意味のある名前を付けます。この名前は、ビデオクラウドスタジオ内に表示されます。
  6. description要素を編集して、プロファイルに意味のある説明を付けます。
  7. 必要に応じて、レンディション情報を編集します。ポスターとサムネイル画像のデフォルトサイズは、プロファイルの末尾にあります。
  8. [ 保存] をクリックします。

プロファイルに透かしを追加する

watermarks各レンディションに要素を追加することで、ウォーターマークを各レンディションに追加できますrendition。

       透かし」: [ { "y": -20, "x": -10, "width": "20%", "url": "http://www.mysite.com/companylogo.png" } ]

watermarks要素には、次のパラメータがあります。

[パラメータ] タイプ 説明
url ひも 透かし画像の URL
width ひも ピクセル単位の幅、またはフレーム幅のパーセント（例: 20 または 10%）
height ひも ピクセル単位の高さまたはフレーム幅のパーセント（例: 20または10％）
x ひも 画像の左端から中心までの距離（ピクセルまたはフレーム幅のパーセント）。例: 20 または 10%（右端に対してウォーターマークを配置するには、-10 などの負の値を使用します）。
y ひも 画像の上端から中央までの距離（ピクセルまたはフレームの高さの割合）。例: 20 または 10%（-10などの負の値を使用して、下端に対してウォーターマークを配置します）
