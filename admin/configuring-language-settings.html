[言語設定] では、プライマリ言語を定義し、追加のメタデータ言語を追加および管理します。主要言語は、アカウントが動画メタデータに現在使用している言語です。アカウントに言語を追加する前に、既存のメタデータは、プライマリ言語として定義した言語に関連付けられます。プライマリ言語は、Media モジュールでビデオプロパティを編集するときに最初に表示される言語です。

プライマリ言語の設定

プライマリ言語を設定するには、次の手順に従います。

  1. ビデオクラウドスタジオにログインします。
  2. ナビゲーションヘッダーで、[ 管理 ]、[ 言語設定] の順にクリックします。管理者メニューにアクセスするには、アカウント管理者である必要があります。
  3. プライマリ言語を入力します。入力すると、一致する言語の一覧が表示されます。
  4. 言語をクリックして選択します。
  5. [ 言語を保存] をクリックします。
プライマリ言語を変更するには、言語をクリックし、新しい言語を選択し、[ 保存] をクリックします。

言語を追加する

プライマリ言語を保存した後、追加のメタデータ言語を追加できます。言語を追加することで、Media モジュールに追加の多言語のメタデータを入力できます。追加の言語を追加するには、次の手順を実行します。

  1. [ 言語を追加] をクリックします。
  2. 言語を入力します。入力すると、一致する言語の一覧が表示されます。
  3. 言語をクリックして選択します。
  4. [ 言語を保存] をクリックします。言語が [追加言語] テーブルに追加されます。
  5. [ 言語を追加] をクリックして、必要に応じて言語を追加します。

メディアモジュールに表示される言語名を編集するには、その言語をクリックして変更し、[ 保存] をクリックします。言語を削除するには、削除アイコン（）。

注:使用中の言語を削除すると、その言語に関連付けられているすべてのメタデータは非表示になります。関連するメタデータを復元するには、言語を再度追加します。
